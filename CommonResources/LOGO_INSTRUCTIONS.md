# 🎨 MathCodeLab Logo Setup Instructions

## Required Logo Files

Please save your logo PNG files in this directory with the following names:

### 1. Dark Logo (for light backgrounds)
- **File name**: `Logo.MathCodeLab.Dark.png`
- **Description**: Logo with dark/black background
- **Usage**: Displays on light-themed pages

### 2. Light Logo (for dark backgrounds)  
- **File name**: `Logo.MathCodeLab.Light.png`
- **Description**: Logo with light/white background
- **Usage**: Displays on dark-themed pages

## File Specifications
- **Format**: PNG
- **Recommended width**: 300-600px
- **Aspect ratio**: Maintain original proportions
- **Transparency**: Supported

## Automatic Theme Switching
The README.md is configured to automatically show:
- **Light logo** on dark themes/backgrounds
- **Dark logo** on light themes/backgrounds

This ensures optimal visibility across all viewing modes!
