## Reversing Numbers & Lists 🔄

### 🧭 Reverse the Train Analogy 🚂

Imagine a train at a station. To reverse it, you disconnect each car from the front and add it to a new train. The last car becomes the first!

### Built-in Reverse
```python
numbers = [1, 2, 3, 4, 5]

# Method 1: Slicing (creates new list)
reversed_list = numbers[::-1]
print(reversed_list)  # [5, 4, 3, 2, 1]

# Method 2: reverse() method (modifies original)
numbers.reverse()
print(numbers)  # [5, 4, 3, 2, 1]

# Method 3: reversed() function
for num in reversed(numbers):
    print(num, end=" ")
```

### Manual Reverse (Two-Pointer Approach)
```python
def reverse_list(lst):
    left = 0
    right = len(lst) - 1
    
    while left < right:
        # Swap elements at left and right pointers
        lst[left], lst[right] = lst[right], lst[left]
        left += 1
        right -= 1
    
    return lst

numbers = [10, 20, 30, 40, 50]
print(reverse_list(numbers))  # [50, 40, 30, 20, 10]
```

**Visualization:**
```
Step 1: [10, 20, 30, 40, 50]
         L               R     → Swap 10 and 50

Step 2: [50, 20, 30, 40, 10]
             L       R         → Swap 20 and 40

Step 3: [50, 40, 30, 20, 10]
                 L=R           → Done! (pointers met)
```

## Reversing Strings 🔤

```python
word = "PYTHON"

# Method 1: Slicing
reversed_word = word[::-1]
print(reversed_word)  # "NOHTYP"

# Method 2: Manual with loop
reversed_word = ""
for char in word:
    reversed_word = char + reversed_word
print(reversed_word)  # "NOHTYP"
```

### Reverse Words in a Sentence
```python
sentence = "I love Python programming"
words = sentence.split()
words.reverse()
reversed_sentence = " ".join(words)
print(reversed_sentence)  # "programming Python love I"
```

## The Two-Pointer Technique 👈👉

The two-pointer technique uses two variables (pointers) that start at different positions and move toward each other.

```
Array: [a, b, c, d, e, f]
        ↑                ↑
       left            right

Move left →    ← Move right
Until they meet in the middle!
```

### Example: Check if Sorted
```python
def is_sorted(lst):
    left = 0
    right = len(lst) - 1
    
    while left < right:
        if lst[left] > lst[left + 1]:
            return False
        left += 1
    
    return True

print(is_sorted([1, 2, 3, 4, 5]))    # True
print(is_sorted([1, 3, 2, 4, 5]))    # False
```

## Palindrome Detection 🪞

A **palindrome** reads the same forwards and backwards: "racecar", "madam", "level"

### Simple Check
```python
word = input("Enter a word: ").lower()

if word == word[::-1]:
    print(f"'{word}' is a palindrome! ✅")
else:
    print(f"'{word}' is NOT a palindrome. ❌")
```

### Two-Pointer Palindrome Check
```python
def is_palindrome(text):
    text = text.lower()
    left = 0
    right = len(text) - 1
    
    while left < right:
        if text[left] != text[right]:
            return False
        left += 1
        right -= 1
    
    return True

print(is_palindrome("racecar"))  # True
print(is_palindrome("hello"))    # False
print(is_palindrome("Madam"))    # True
```

### Number Palindrome
```python
def is_number_palindrome(n):
    text = str(n)
    return text == text[::-1]

print(is_number_palindrome(12321))  # True
print(is_number_palindrome(12345))  # False
```
