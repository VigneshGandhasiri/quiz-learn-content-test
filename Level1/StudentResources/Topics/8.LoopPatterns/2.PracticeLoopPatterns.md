# Loop Patterns: Practice Exercises ✏️

Master nested loops, break, continue, and accumulation patterns!

---

## 1. Star Rectangle

```python
rows = 4
cols = 6
for r in range(rows):
    for c in range(cols):
        print("*", end="")
    print()
```

---

## 2. Skip Multiples of 3

```python
for i in range(1, 21):
    if i % 3 == 0:
        continue
    print(i, end=" ")
print()
```

## Practice Exercises

1. **Right Triangle** — Print a right triangle of stars with n rows. Row 1 has 1 star, row 2 has 2, etc.

2. **Number Grid** — Print a grid where each cell shows `row * col`. For example, a 5×5 grid is a mini multiplication table.

3. **Find First Multiple** — Given a number, use a loop with `break` to find the first multiple of that number greater than 100.

4. **Sum Until Limit** — Read numbers one at a time. Keep a running sum. Stop (break) when the sum exceeds 1000 and print how many numbers were read.

5. **Even Stars, Odd Dashes** — Print n lines. Even-numbered lines get stars `***`, odd-numbered lines get dashes `---`.

---

## Challenge Exercises 🌟

6. **Diamond Shape** — Print a diamond of stars given an odd number n for the widest row.

7. **Spiral Numbers** — Print numbers 1 to n×n arranged in a spiral pattern in an n×n grid.
