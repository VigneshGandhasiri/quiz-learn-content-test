# Array Math: Practice Exercises ✏️

Practice running sums, prefix sums, and accumulation patterns!

---

## 1. Running Sum Example

```python
nums = [1, 3, 5, 2, 4]
running = []
total = 0
for n in nums:
    total += n
    running.append(total)
print(running)  # [1, 4, 9, 11, 15]
```

---

## 2. Prefix Sum Range Query

```python
nums = [2, 4, 6, 8, 10]
prefix = [0] * (len(nums) + 1)
for i in range(len(nums)):
    prefix[i + 1] = prefix[i] + nums[i]

# Sum from index 1 to 3: prefix[4] - prefix[1]
print(prefix[4] - prefix[1])  # 18 (4+6+8)
```

## Practice Exercises

1. **Running Sum** — Given a list of integers, compute the running sum and print it.

2. **Running Average** — Given a list of scores, print the running average after each score is added (round to 2 decimal places).

3. **Range Sum Queries** — Given a list and several queries [L, R], compute the sum of elements from index L to R using prefix sums.

4. **Maximum Subarray Sum** — Find the contiguous subarray with the maximum sum (Kadane's algorithm).

5. **Difference Array** — Given a list, compute the difference between each adjacent pair of elements.
