## Min & Max 📉📈

### Built-in Functions
```python
temperatures = [72, 85, 68, 91, 77, 63, 88]

print(f"Hottest: {max(temperatures)}°F")    # 91
print(f"Coldest: {min(temperatures)}°F")    # 63
```

### Finding Min/Max Manually (understanding the algorithm!)
```python
temps = [72, 85, 68, 91, 77, 63, 88]

# Find minimum manually
current_min = temps[0]
for temp in temps:
    if temp < current_min:
        current_min = temp
print(f"Min: {current_min}")  # 63

# Find maximum manually
current_max = temps[0]
for temp in temps:
    if temp > current_max:
        current_max = temp
print(f"Max: {current_max}")  # 91
```

## Sum & Average 📊

### 🧭 Temperature Tracker Analogy 🌡️

Imagine tracking daily temperatures for a week and computing the weekly average:

```python
daily_temps = [72, 85, 68, 91, 77, 63, 88]

total = sum(daily_temps)
count = len(daily_temps)
average = total / count

print(f"Total: {total}°F")
print(f"Days: {count}")
print(f"Average: {average:.1f}°F")  # 77.7°F
```

### Manual Sum
```python
numbers = [10, 20, 30, 40, 50]
total = 0
for num in numbers:
    total += num
print(f"Sum: {total}")  # 150
```

## Finding Missing Numbers 🔍

Given a sequence that should be continuous, find what's missing:

```python
# Should be 1-10 but one number is missing
numbers = [1, 2, 3, 4, 6, 7, 8, 9, 10]

expected_sum = sum(range(1, 11))   # 55
actual_sum = sum(numbers)           # 50
missing = expected_sum - actual_sum

print(f"Missing number: {missing}")  # 5
```

### Finding Multiple Missing Numbers
```python
full_set = set(range(1, 11))
given = set([1, 2, 4, 5, 7, 9, 10])
missing = full_set - given
print(f"Missing: {sorted(missing)}")  # [3, 6, 8]
```

## Filtering & Counting 🧮

### Count elements matching a condition
```python
scores = [85, 42, 91, 73, 58, 95, 67, 88]

passing = 0
for score in scores:
    if score >= 70:
        passing += 1

print(f"{passing} out of {len(scores)} passed")  # 5 out of 8 passed
```

### Filter into a new list
```python
ages = [12, 17, 8, 21, 15, 9, 18]

teens = []
for age in ages:
    if 13 <= age <= 19:
        teens.append(age)

print(f"Teens: {teens}")  # [17, 15, 18]
```
