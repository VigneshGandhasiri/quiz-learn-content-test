# Traversal Patterns: Practice Exercises ✏️

Master row, column, diagonal, and boundary traversals on 2D grids!

---

## 1. Diagonal Traversal

```python
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
# Main diagonal: matrix[i][i]
for i in range(len(matrix)):
    print(matrix[i][i], end=" ")
print()  # 1 5 9
```

---

## 2. Boundary Walk

```python
grid = [[1,2,3],[4,5,6],[7,8,9]]
n = len(grid)
# Top row, right col, bottom row reversed, left col reversed
boundary = grid[0] + [grid[i][n-1] for i in range(1,n)] + grid[n-1][-2::-1] + [grid[i][0] for i in range(n-2,0,-1)]
print(boundary)  # [1,2,3,6,9,8,7,4]
```
