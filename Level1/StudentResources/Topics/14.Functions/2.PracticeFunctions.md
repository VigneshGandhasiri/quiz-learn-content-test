# Functions: Practice Exercises ✏️

Practice defining, calling, and composing Python functions!

---

## 1. Basic Function

```python
def greet(name):
    return f"Hello, {name}! Welcome!"

message = greet("Alice")
print(message)
```

---

## 2. Function with Default Parameter

```python
def power(base, exp=2):
    return base ** exp

print(power(3))     # 9  (3²)
print(power(2, 5))  # 32 (2⁵)
```

## Practice Exercises

1. **Temperature Converter** — Write two functions: `celsius_to_fahrenheit(c)` and `fahrenheit_to_celsius(f)`. Test them with several values.

2. **Is Even/Odd** — Write a function `is_even(n)` that returns True if n is even. Use it in a loop to classify numbers 1-20.

3. **Factorial** — Write a function `factorial(n)` that computes n! using a loop. Test with 0, 1, 5, 10.

4. **Min of Three** — Write a function `min_of_three(a, b, c)` that returns the smallest of three numbers WITHOUT using `min()`.

5. **Password Validator** — Write a function that checks if a password is at least 8 characters and contains at least one digit. Return True/False.

## Challenge Exercises 🌟

6. **Function Composition** — Write `compose(f, g)` that returns a new function applying g first, then f. Example: `compose(double, add_one)(5)` → 12.

7. **Recursive Fibonacci** — Write a recursive function to compute the nth Fibonacci number. Compare with an iterative version.
