# 🔢 Data Types — What Kind of Data Is This?

Not all data is the same! A name is different from a number, and a number is different from a True/False answer. Python needs to know what **type** of data it's working with — and so do you!

## Why Data Types Matter 🤔

Think of a **shape sorter toy** 🔷🔶🟢 — you can't fit a square block through a round hole! Data types work the same way: you can't do math with text, and you can't spell words with numbers.

```python
# These look similar but behave VERY differently!
print(5 + 5)       # 10  (number addition)
print("5" + "5")   # "55" (text joining!)
```

## The Four Core Types 🏗️

### 1. `int` — Integer (Whole Numbers)
```python
age = 14
score = 100
temperature = -5
print(type(age))   # <class 'int'>
```

### 2. `float` — Decimal Numbers
```python
price = 9.99
pi = 3.14159
grade = 85.5
print(type(price))  # <class 'float'>
```

### 3. `str` — String (Text)
```python
name = "Alex"
greeting = 'Hello, World!'
empty = ""
print(type(name))   # <class 'str'>
```

### 4. `bool` — Boolean (True/False)
```python
is_student = True
has_passed = False
print(type(is_student))  # <class 'bool'>
```

### 🧭 Menu Data Analogy 🍔

Think of a restaurant menu:
| Menu Item | Data Type | Example |
|-----------|-----------|---------|
| Item name | `str` | `"Cheeseburger"` |
| Price | `float` | `8.99` |
| Quantity available | `int` | `25` |
| Is spicy? | `bool` | `True` |

## Type Checking 🔍

Use `type()` to check what type a value is:

```python
print(type(42))         # <class 'int'>
print(type(3.14))       # <class 'float'>
print(type("hello"))    # <class 'str'>
print(type(True))       # <class 'bool'>
```

### Quick Quiz: What type is each?
```python
a = 100        # int
b = "100"      # str ← quotes make it text!
c = 100.0      # float ← decimal point makes it float!
d = True       # bool
```

## Type Casting 🔄

**Type casting** means converting one type to another.

### String to Number
```python
age_text = "14"
age_number = int(age_text)     # Convert to int: 14
price_text = "9.99"
price_number = float(price_text)  # Convert to float: 9.99
```

### Number to String
```python
score = 95
score_text = str(score)        # Convert to string: "95"
print("Your score: " + score_text)
```

### Integer ↔ Float
```python
x = int(9.7)      # 9 (drops decimal, does NOT round!)
y = float(5)       # 5.0
```

### Casting Diagram
```
     int()          float()          str()
"14" ────▶ 14      14 ────▶ 14.0    14 ────▶ "14"
"3.5" ──▶ ERROR    "3.5" ──▶ 3.5    3.5 ──▶ "3.5"
```

## Common Type Errors ❌

### Error 1: Adding string + number
```python
name = "Alex"
age = 14
# print(name + age)  ❌ TypeError!

# Fix: Convert number to string
print(name + str(age))  # ✅ "Alex14"
# Or use comma:
print(name, age)         # ✅ "Alex 14"
```

### Error 2: Math with strings
```python
a = "10"
b = "5"
# print(a - b)  ❌ TypeError! Can't subtract strings!

# Fix: Convert to numbers first
print(int(a) - int(b))  # ✅ 5
```

### Error 3: Invalid conversion
```python
# int("hello")  ❌ ValueError! "hello" is not a number!
# int("3.14")   ❌ ValueError! Use float() first!

x = float("3.14")  # ✅ 3.14
y = int(x)          # ✅ 3
```

## String Operations 🔤

Strings have their own special operations:

```python
# Concatenation (joining)
first = "Hello"
second = "World"
print(first + " " + second)  # "Hello World"

# Repetition
print("Ha" * 3)    # "HaHaHa"
print("-" * 20)    # "--------------------"

# Length
message = "Python is fun!"
print(len(message))  # 14
```
