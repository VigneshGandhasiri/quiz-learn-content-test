# Data Types: Practice Exercises ✏️

Practice identifying, converting, and working with different data types!

---

## 1. Type Detective

```python
a = 42
b = "42"
c = 42.0
d = True
print(type(a))  # <class 'int'>
print(type(b))  # <class 'str'>
print(type(c))  # <class 'float'>
print(type(d))  # <class 'bool'>
```

---

## 2. Type Conversion

```python
num_str = "25"
num_int = int(num_str)
num_float = float(num_str)
print(num_int + 10)     # 35
print(num_float + 0.5)  # 25.5
```

---

## 3. String Math vs Number Math

```python
print(3 + 4)          # 7
print("3" + "4")      # "34"
print(str(3) + str(4))  # "34"
print(int("3") + int("4"))  # 7
```

## Practice Exercises

1. **Type Identifier** — Given these values, predict the type of each WITHOUT running the code: `"True"`, `3.0`, `0`, `""`, `False`, `"3.14"`

2. **Unit Price Calculator** — Ask the user for total cost (float) and quantity (int). Calculate and display the unit price.

3. **Receipt Builder** — Create variables for item name (str), quantity (int), price (float), and taxable (bool). Print a formatted receipt line.

4. **String Repeater** — Ask the user for a word and a number. Print the word repeated that many times. (Hint: you'll need type conversion!)

5. **Debug Challenge** — Fix this broken code:
   ```python
   age = input("Your age: ")
   next_year = age + 1
   print("Next year you'll be", next_year)
   ```
