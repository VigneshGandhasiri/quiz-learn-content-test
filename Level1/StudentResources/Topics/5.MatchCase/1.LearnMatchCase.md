# 🎰 Match-Case — Pattern Matching Made Clean

When you have many specific values to check, using lots of `if-elif` can get messy. Python's `match-case` statement (introduced in Python 3.10) provides a **cleaner, more readable way** to handle multiple specific options!

## Why Match-Case? 🤔

### 🧭 Vending Machine Analogy 🥤

Imagine a vending machine with buttons:
- Press **A1** → Get Cola
- Press **A2** → Get Sprite
- Press **B1** → Get Chips
- Press **B2** → Get Cookies
- Press **anything else** → "Invalid selection!"

Each button maps **directly** to one specific result. That's exactly what `match-case` does!

```
Button ──▶ match
  A1   ──▶ case "A1": Cola
  A2   ──▶ case "A2": Sprite
  B1   ──▶ case "B1": Chips
  ???  ──▶ case _: Invalid!
```

---

## Basic match-case Syntax 📝

```python
command = input("Enter command: ")

match command:
    case "start":
        print("Starting the game...")
    case "pause":
        print("Game paused.")
    case "quit":
        print("Goodbye!")
    case _:
        print("Unknown command!")
```

### Structure Breakdown:
```python
match variable:          # What are we checking?
    case value1:         # If variable == value1
        action1          # Do this
    case value2:         # If variable == value2
        action2          # Do this
    case _:              # If nothing else matched
        default_action   # Do this (catch-all)
```

---

## The Default Case `_` 🔄

The `_` (underscore) is the **wildcard** — it matches **anything** that wasn't caught by previous cases.

```python
day_number = int(input("Enter day number (1-7): "))

match day_number:
    case 1:
        print("Monday")
    case 2:
        print("Tuesday")
    case 3:
        print("Wednesday")
    case 4:
        print("Thursday")
    case 5:
        print("Friday")
    case 6:
        print("Saturday")
    case 7:
        print("Sunday")
    case _:
        print("Invalid day number!")
```

### ⚠️ Always Include `_`!
Without the default case, unexpected inputs will silently do nothing. Always have a catch-all!

## Real-World Applications 🌍

### 🧭 TV Remote Analogy 📺

```python
button = input("Press a button: ")

match button:
    case "power":
        print("TV turning on/off...")
    case "volume_up":
        print("Volume: 🔊 Louder!")
    case "volume_down":
        print("Volume: 🔉 Quieter!")
    case "mute":
        print("Volume: 🔇 Muted!")
    case "channel_up":
        print("Channel ⬆️")
    case "channel_down":
        print("Channel ⬇️")
    case _:
        print("Unknown button!")
```

### Calculator Menu

```python
print("=== Calculator ===")
print("1. Add")
print("2. Subtract")
print("3. Multiply")
print("4. Divide")

choice = input("Choose operation (1-4): ")
a = float(input("First number: "))
b = float(input("Second number: "))

match choice:
    case "1":
        print(f"Result: {a + b}")
    case "2":
        print(f"Result: {a - b}")
    case "3":
        print(f"Result: {a * b}")
    case "4":
        if b != 0:
            print(f"Result: {a / b}")
        else:
            print("Error: Cannot divide by zero!")
    case _:
        print("Invalid choice!")
```

### Season Finder

```python
month = input("Enter month name: ").lower()

match month:
    case "december" | "january" | "february":
        print("❄️ Winter")
    case "march" | "april" | "may":
        print("🌸 Spring")
    case "june" | "july" | "august":
        print("☀️ Summer")
    case "september" | "october" | "november":
        print("🍂 Fall")
    case _:
        print("That's not a valid month!")
```

### 🎯 Using `|` for Multiple Matches
The `|` operator lets you combine multiple values in a single case!

## Match-Case vs if-elif 🆚

### When to use `match-case`:
- Checking a single variable against **specific values**
- Menu systems and command handlers
- Mapping inputs to outputs directly

### When to use `if-elif`:
- Checking **ranges** of values (e.g., score >= 90)
- Combining multiple conditions with `and`/`or`
- Complex boolean logic

```python
# ✅ match-case is BETTER here:
match color:
    case "red":    print("Stop!")
    case "yellow": print("Caution!")
    case "green":  print("Go!")

# ✅ if-elif is BETTER here:
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
```

## 🎯 Key Takeaways

| Concept | Description | Example |
|---------|-------------|---------|
| `match` | Start pattern matching | `match command:` |
| `case value:` | Check for specific value | `case "start":` |
| `case _:` | Default/catch-all case | Handles everything else |
| `case a \| b:` | Match multiple values | `case "y" \| "yes":` |
