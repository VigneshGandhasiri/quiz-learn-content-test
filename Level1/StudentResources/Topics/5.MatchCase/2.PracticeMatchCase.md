# Match-Case: Practice Exercises ✏️

Practice using pattern matching to build clean, readable programs!

---

## 1. Simple Command Handler

```python
command = input("Enter command: ").lower()
match command:
    case "hello":
        print("Hi there!")
    case "bye":
        print("See you later!")
    case _:
        print("I don't understand that command.")
```

---

## 2. Emoji Translator

```python
word = input("Enter a word: ").lower()
match word:
    case "happy":
        print("😊")
    case "sad":
        print("😢")
    case "angry":
        print("😠")
    case "love":
        print("❤️")
    case _:
        print("🤷 No emoji for that!")
```

## Practice Exercises

1. **Direction Mover** — Ask the user to type "north", "south", "east", or "west". Print "Moving [direction]!" or "Invalid direction!" for anything else.

2. **Grade to Feedback** — Given a letter grade (A, B, C, D, F), use match-case to print appropriate feedback.

3. **Coin Converter** — Ask for a coin name (penny, nickel, dime, quarter). Use match-case to print its value in cents.

4. **Rock Paper Scissors** — Rewrite Rock Paper Scissors using match-case instead of if-elif. (Compare how much cleaner it looks!)

5. **Month Days** — Given a month number (1-12), print how many days that month has. Use `|` to group months with the same number of days.

---

## Challenge Exercises 🌟

6. **Multi-Language Greeter** — Ask for a language (english, spanish, french, japanese, etc.). Print "Hello" in that language. Support at least 5 languages.

7. **Roman Numeral Converter** — Convert a single Roman numeral character (I, V, X, L, C, D, M) to its decimal value using match-case.
