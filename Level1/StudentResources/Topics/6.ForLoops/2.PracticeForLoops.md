# For Loops: Practice Exercises ✏️

Practice using for loops and range() to solve repetition problems!

---

## 1. Simple Counter

```python
for i in range(1, 11):
    print(i)
print("Done counting!")
```

---

## 2. Multiplication Table

```python
num = int(input("Enter a number: "))
for i in range(1, 11):
    print(f"{num} x {i} = {num * i}")
```

## Practice Exercises

1. **Sum 1 to N** — Ask the user for a number N. Print the sum of all numbers from 1 to N.

2. **Even Numbers** — Print all even numbers from 2 to 50 using range() with a step.

3. **Star Pyramid** — Print this pattern using a loop:
   ```
   *
   **
   ***
   ****
   *****
   ```

4. **Reverse Countdown** — Print numbers from 20 down to 1.

5. **Character Counter** — Ask for a word and a letter. Count how many times that letter appears in the word.
