- **Binary search** requires sorted data
- **Finding duplicates** is easier with sorted data
- **Displaying rankings** (leaderboards, grades)
- **Organizing databases** for efficient access

## Bubble Sort 🫧

**Idea**: Repeatedly walk through the list, compare adjacent pairs, and swap if they're in the wrong order. Larger values "bubble up" to the end!

```python
def bubble_sort(lst):
    n = len(lst)
    for i in range(n - 1):
        for j in range(n - 1 - i):
            if lst[j] > lst[j + 1]:
                lst[j], lst[j + 1] = lst[j + 1], lst[j]  # Swap!
    return lst

numbers = [64, 34, 25, 12, 22, 11, 90]
print(bubble_sort(numbers))  # [11, 12, 22, 25, 34, 64, 90]
```

### Visualization
```
Pass 1: [64, 34, 25, 12] → [34, 25, 12, 64]  (64 bubbles to end)
Pass 2: [34, 25, 12, 64] → [25, 12, 34, 64]  (34 bubbles up)
Pass 3: [25, 12, 34, 64] → [12, 25, 34, 64]  (25 bubbles up) ✅
```

## Selection Sort 🎯

**Idea**: Find the **smallest element** and put it first. Then find the next smallest and put it second. Repeat!

```python
def selection_sort(lst):
    n = len(lst)
    for i in range(n):
        min_idx = i
        for j in range(i + 1, n):
            if lst[j] < lst[min_idx]:
                min_idx = j
        lst[i], lst[min_idx] = lst[min_idx], lst[i]
    return lst

numbers = [64, 25, 12, 22, 11]
print(selection_sort(numbers))  # [11, 12, 22, 25, 64]
```

### Visualization
```
[64, 25, 12, 22, 11]  → Find min (11), swap with first
[11, 25, 12, 22, 64]  → Find min of rest (12), swap with second
[11, 12, 25, 22, 64]  → Find min of rest (22), swap with third
[11, 12, 22, 25, 64]  → Find min of rest (25), already in place
[11, 12, 22, 25, 64]  ✅ Sorted!
```

## Insertion Sort 🃏

**Idea**: Build the sorted list one element at a time — like sorting cards in your hand! Pick up each card and insert it in the correct position.

```python
def insertion_sort(lst):
    for i in range(1, len(lst)):
        key = lst[i]
        j = i - 1
        while j >= 0 and lst[j] > key:
            lst[j + 1] = lst[j]
            j -= 1
        lst[j + 1] = key
    return lst

numbers = [12, 11, 13, 5, 6]
print(insertion_sort(numbers))  # [5, 6, 11, 12, 13]
```

### Visualization
```
[12 | 11, 13, 5, 6]  → Insert 11 → [11, 12 | 13, 5, 6]
[11, 12 | 13, 5, 6]  → Insert 13 → [11, 12, 13 | 5, 6]
[11, 12, 13 | 5, 6]  → Insert 5  → [5, 11, 12, 13 | 6]
[5, 11, 12, 13 | 6]  → Insert 6  → [5, 6, 11, 12, 13] ✅
```

## Comparing Sort Algorithms 📊

| Algorithm | Best Case | Worst Case | Stable? | When to Use |
|-----------|-----------|------------|---------|-------------|
| Bubble Sort | O(n) | O(n²) | Yes | Already nearly sorted |
| Selection Sort | O(n²) | O(n²) | No | Small lists, minimal swaps |
| Insertion Sort | O(n) | O(n²) | Yes | Small or nearly sorted |

### Python's Built-in Sort
```python
numbers = [3, 1, 4, 1, 5, 9, 2, 6]
numbers.sort()                    # Sorts in-place
print(numbers)                    # [1, 1, 2, 3, 4, 5, 6, 9]

sorted_nums = sorted([3, 1, 4])   # Returns new sorted list
print(sorted_nums)                 # [1, 3, 4]
```
Python uses **Timsort** (O(n log n)) — much more efficient than our three algorithms!
