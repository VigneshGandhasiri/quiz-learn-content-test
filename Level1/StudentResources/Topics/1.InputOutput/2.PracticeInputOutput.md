# Input & Output: Practice Exercises ✏️

Practice using `print()` and `input()` to build programs that communicate!

---

## 1. Hello World

```python
# Your very first program!
print("Hello, World!")
```

---

## 2. Personal Greeting

```python
name = input("What is your name? ")
print("Nice to meet you,", name + "!")
```

---

## 3. Age Calculator

```python
birth_year = int(input("What year were you born? "))
current_year = 2026
age = current_year - birth_year
print("You are approximately", age, "years old!")
```

## Practice Exercises

Write small programs using `print()` and `input()`:

1. **Self Introduction** — Ask the user for their name, favorite color, and hobby. Print a paragraph about them.

2. **Rectangle Area** — Ask for length and width, calculate and display the area.

3. **Temperature Reporter** — Ask for a temperature in Fahrenheit. Print: "Today's temperature is __ degrees Fahrenheit."

4. **Three Friends** — Ask for three friends' names. Print them in a list format:
   ```
   My friends are:
   1. Alice
   2. Bob
   3. Charlie
   ```

5. **Story Builder** — Ask for a noun, verb, adjective, and place. Build a fun sentence with them.

---

## Challenge Exercises 🌟

6. **Tip Calculator** — Ask for a restaurant bill amount. Calculate and display a 15% tip and the total.

7. **Miles to Kilometers** — Ask for a distance in miles. Convert to kilometers (multiply by 1.609) and display both values.

8. **Multi-line Art** — Use multiple `print()` calls to create a simple ASCII art picture (like a house or a tree).
