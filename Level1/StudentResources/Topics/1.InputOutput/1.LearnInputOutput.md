## The Black Box Model 📦

A **Black Box** is something that:
1. **Receives input** (data goes IN)
2. **Processes it** (something happens INSIDE)
3. **Produces output** (results come OUT)

```
┌──────────┐     ┌─────────────┐     ┌──────────┐
│  INPUT   │ ──▶ │   PROCESS   │ ──▶ │  OUTPUT  │
│ (data)   │     │  (program)  │     │ (result) │
└──────────┘     └─────────────┘     └──────────┘
```

### 🧭 Real-World Examples

| System | Input | Process | Output |
|--------|-------|---------|--------|
| 🍕 Pizza Oven | Raw dough + toppings | Baking at 450°F | Delicious pizza |
| 🔢 Calculator | Numbers + operation | Math calculation | Answer |
| 🌡️ Thermometer | Temperature reading | Convert to display | Number on screen |
| 🎮 Video Game | Button press | Game logic | Character moves |

## Output with `print()` 🖨️

The `print()` function is how your program **talks back to you**. It displays text on the screen.

### Basic Print

```python
print("Hello, World!")
print("My name is Python!")
print("I can count: 1, 2, 3!")
```

**Output:**
```
Hello, World!
My name is Python!
I can count: 1, 2, 3!
```

### Printing Numbers

```python
print(42)
print(3.14)
print(100 + 200)
```

**Output:**
```
42
3.14
300
```

### Printing Multiple Things

```python
print("I am", 12, "years old")
print("Score:", 95, "out of", 100)
```

**Output:**
```
I am 12 years old
Score: 95 out of 100
```

### Empty Lines

```python
print("Line 1")
print()
print("Line 3")
```

**Output:**
```
Line 1

Line 3
```

## Input with `input()` ⌨️

The `input()` function lets your program **ask the user a question** and wait for their answer.

### Basic Input

```python
name = input("What is your name? ")
print("Hello,", name)
```

**What happens:**
```
What is your name? Alex
Hello, Alex
```

### 🎯 Important Rule!
`input()` **always returns text** (a string). Even if the user types a number, Python treats it as text!

```python
age = input("How old are you? ")
print("You said:", age)
print("Type of age:", type(age))
```

**Output:**
```
How old are you? 14
You said: 14
Type of age: <class 'str'>
```

### Converting Input to Numbers

```python
age = int(input("How old are you? "))     # Convert to whole number
price = float(input("Enter price: $"))    # Convert to decimal number
```

## Putting It All Together 🔗

### Example 1: Greeting Program

```python
name = input("Enter your name: ")
age = input("Enter your age: ")
print("Welcome,", name + "!")
print("You are", age, "years old.")
```

### Example 2: Simple Calculator

```python
num1 = int(input("Enter first number: "))
num2 = int(input("Enter second number: "))
total = num1 + num2
print("The sum is:", total)
```

### Example 3: Mad Libs!

```python
animal = input("Enter an animal: ")
food = input("Enter a food: ")
place = input("Enter a place: ")
print("The", animal, "ate", food, "at the", place + "!")
```

## Systems Thinking 🧠

### Garbage In → Garbage Out (GIGO)

If you give a program **bad input**, you get **bad output**. This is one of the most important rules in programming!

```python
# What happens if someone types "hello" when we expect a number?
num = int(input("Enter a number: "))  # ❌ Crashes if user types "hello"!
```

### 🤔 Think About This...
- What if someone enters their age as "one hundred" instead of "100"?
- What if someone enters a negative number for their age?
- What if someone leaves the input blank?

Good programmers **always think about what could go wrong** with their input!

## 🎯 Key Takeaways

| Concept | What It Does | Example |
|---------|-------------|---------|
| `print()` | Shows output on screen | `print("Hello!")` |
| `input()` | Gets text from user | `name = input("Name? ")` |
| `int()` | Converts to whole number | `int("42")` → `42` |
| `float()` | Converts to decimal | `float("3.14")` → `3.14` |
| Black Box | Input → Process → Output | Every program! |
| GIGO | Bad input = Bad output | Always validate! |
