# Queues & Stacks: Practice Exercises ✏️

Practice FIFO and LIFO data structures!

---

## 1. Stack with a List

```python
stack = []
stack.append("A")  # push
stack.append("B")
stack.append("C")
print(stack.pop())  # C (LIFO)
print(stack.pop())  # B
print(stack)        # ['A']
```

---

## 2. Queue with deque

```python
from collections import deque
queue = deque()
queue.append("first")   # enqueue
queue.append("second")
queue.append("third")
print(queue.popleft())  # first (FIFO)
print(queue)            # deque(['second', 'third'])
```

## Practice Exercises

1. **Undo System** — Use a stack to implement an undo system. User types text commands; "undo" pops the last command. Print the remaining stack.

2. **Print Queue** — Simulate a printer queue. Jobs arrive with priorities 1-3. Process jobs in FIFO order and print each one as it's "printed".

3. **Balanced Parentheses** — Check if a string of parentheses `()[]{}` is balanced using a stack.

4. **Reverse a String** — Use a stack to reverse a string character by character.

5. **Hot Potato** — Simulate the hot potato game: n people stand in a circle (queue). Every k-th person is removed until one remains.
