# 📚 Queues & Stacks — Ordered Data Structures

Queues and stacks are two fundamental ways to organize data where the **order of operations matters**. They appear everywhere: from printer jobs to browser history!

---

## Part 1: Queues 🚶‍♂️🚶‍♂️🚶‍♂️ (FIFO)

### What Is a Queue?
A queue follows **FIFO** — **First In, First Out**. Just like a real line at a store!

```
ENQUEUE (add to back)              DEQUEUE (remove from front)
        ↓                                 ↓
Back → [D] [C] [B] [A] → Front
```

### 🧭 Dragon's Queue Analogy 🐉
Imagine brave warriors lining up to fight a dragon. The first warrior who arrived fights first!

```python
from collections import deque

queue = deque()

# Enqueue (add to back)
queue.append("Knight")
queue.append("Wizard")
queue.append("Archer")
print(f"Queue: {list(queue)}")  # ['Knight', 'Wizard', 'Archer']

# Dequeue (remove from front)
fighter = queue.popleft()
print(f"{fighter} fights the dragon!")  # Knight fights the dragon!
print(f"Queue: {list(queue)}")  # ['Wizard', 'Archer']
```

### 🧭 Lunch Line Analogy 🍽️
```python
lunch_line = deque()

# Students arrive
lunch_line.append("Alex")
lunch_line.append("Sam")
lunch_line.append("Jordan")

# Students get served (first arrives, first served)
while lunch_line:
    student = lunch_line.popleft()
    print(f"Serving {student}! 🍽️")
```

### Queue Using a List (Simple Version)
```python
queue = []
queue.append("task1")      # Enqueue
queue.append("task2")
queue.append("task3")

first = queue.pop(0)        # Dequeue (less efficient than deque)
print(first)                # "task1"
```

## Part 2: Stacks 📚 (LIFO)

### What Is a Stack?
A stack follows **LIFO** — **Last In, First Out**. Like a stack of plates — you take from the top!

```
PUSH (add to top)
        ↓
      [  D  ]  ← TOP (last added, first removed)
      [  C  ]
      [  B  ]
      [  A  ]  ← BOTTOM (first added, last removed)
```

### 🧭 Pancake Tower Analogy 🥞
You stack pancakes on a plate. The last pancake you put on top is the first one you eat!

```python
pancake_stack = []

# Push (add to top)
pancake_stack.append("Blueberry")
pancake_stack.append("Chocolate")
pancake_stack.append("Plain")

print(f"Stack: {pancake_stack}")

# Pop (remove from top)
top = pancake_stack.pop()
print(f"Eating: {top}")        # Plain (last added)
top = pancake_stack.pop()
print(f"Eating: {top}")        # Chocolate
```

### Undo/Redo System
```python
actions = []

actions.append("Type 'Hello'")
actions.append("Type ' World'")
actions.append("Delete 'd'")

# Undo last action
undone = actions.pop()
print(f"Undone: {undone}")     # Delete 'd'
print(f"Remaining: {actions}") # ["Type 'Hello'", "Type ' World'"]
```

### Balanced Parentheses Checker
```python
def is_balanced(expression):
    stack = []
    pairs = {"(": ")", "[": "]", "{": "}"}
    
    for char in expression:
        if char in pairs:           # Opening bracket
            stack.append(char)
        elif char in pairs.values(): # Closing bracket
            if not stack:
                return False
            top = stack.pop()
            if pairs[top] != char:
                return False
    
    return len(stack) == 0

print(is_balanced("(())"))      # True
print(is_balanced("([{}])"))    # True
print(is_balanced("(()"))       # False
print(is_balanced(")("))        # False
```

### 🧭 Mirror Word Checker 🪞
Use a stack to check if a word reads the same forwards and backwards:

```python
def is_mirror(word):
    stack = []
    
    # Push all characters onto stack
    for char in word:
        stack.append(char)
    
    # Pop and compare
    for char in word:
        if char != stack.pop():
            return False
    return True

print(is_mirror("racecar"))  # True
print(is_mirror("hello"))    # False
```

## Queue vs Stack Summary 🆚

| Feature | Queue (FIFO) | Stack (LIFO) |
|---------|-------------|-------------|
| Add | `append()` (back) | `append()` (top) |
| Remove | `popleft()` (front) | `pop()` (top) |
| Example | Waiting line | Stack of plates |
| Use case | Task scheduling | Undo/Redo |
