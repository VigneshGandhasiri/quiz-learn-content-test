## 1. Create and Print a Grid

```python
rows, cols = 3, 4
grid = []
for r in range(rows):
    row = []
    for c in range(cols):
        row.append(r * cols + c + 1)
    grid.append(row)

for row in grid:
    print(row)
```

## 2. Matrix Addition

```python
a = [[1, 2], [3, 4]]
b = [[5, 6], [7, 8]]
result = []
for i in range(len(a)):
    row = []
    for j in range(len(a[0])):
        row.append(a[i][j] + b[i][j])
    result.append(row)
print(result)  # [[6, 8], [10, 12]]
```
