# 🗂️ Sets & Dictionaries — Smart Data Storage

Sets and dictionaries are powerful data structures that solve specific problems much better than lists. **Sets** handle uniqueness and membership. **Dictionaries** handle key-value mapping.

---

## Part 1: Sets 🔵

### What Is a Set?
A set is an **unordered collection of unique elements** — no duplicates allowed!

```python
fruits = {"apple", "banana", "cherry"}
print(fruits)          # Order may vary!
print(len(fruits))     # 3
```

### Duplicate Detection
```python
numbers = [1, 2, 3, 2, 4, 3, 5, 1]
unique = set(numbers)
print(unique)          # {1, 2, 3, 4, 5}
print(f"Duplicates removed: {len(numbers) - len(unique)}")  # 3
```

### Membership Checking (Super Fast!)
```python
valid_colors = {"red", "green", "blue", "yellow"}
color = input("Enter a color: ").lower()

if color in valid_colors:    # Much faster than checking a list!
    print("Valid color! ✅")
else:
    print("Invalid color! ❌")
```

### Set Operations
```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a | b)    # Union: {1, 2, 3, 4, 5, 6}
print(a & b)    # Intersection: {3, 4}
print(a - b)    # Difference: {1, 2}
```

## Part 2: Dictionaries 📖

### What Is a Dictionary?
A dictionary stores **key-value pairs** — look up a value using its key!

```python
student = {
    "name": "Alex",
    "age": 14,
    "grade": "A",
    "enrolled": True
}

print(student["name"])     # "Alex"
print(student["age"])      # 14
```

### Creating & Modifying
```python
scores = {}
scores["math"] = 95
scores["science"] = 88
scores["english"] = 92

print(scores)  # {"math": 95, "science": 88, "english": 92}

# Update a value
scores["math"] = 97

# Check if key exists
if "art" in scores:
    print(scores["art"])
else:
    print("No art score")
```

### Frequency Counting
```python
text = "hello world"
freq = {}
for char in text:
    if char in freq:
        freq[char] += 1
    else:
        freq[char] = 1
print(freq)  # {'h': 1, 'e': 1, 'l': 3, 'o': 2, ' ': 1, 'w': 1, 'r': 1, 'd': 1}
```

### Looping Through Dictionaries
```python
student = {"name": "Alex", "age": 14, "grade": "A"}

# Loop through keys
for key in student:
    print(f"{key}: {student[key]}")

# Loop through key-value pairs
for key, value in student.items():
    print(f"{key} = {value}")
```
