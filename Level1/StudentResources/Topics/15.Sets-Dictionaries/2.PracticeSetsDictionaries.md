# Sets & Dictionaries: Practice Exercises ✏️

Practice unique collections and key-value lookups!

## 1. Set Operations

```python
a = {1, 2, 3, 4, 5}
b = {4, 5, 6, 7, 8}
print(a & b)  # {4, 5} — intersection
print(a | b)  # {1,2,3,4,5,6,7,8} — union
print(a - b)  # {1, 2, 3} — difference
```

---

## 2. Dictionary Frequency Counter

```python
words = "the cat sat on the mat".split()
freq = {}
for w in words:
    freq[w] = freq.get(w, 0) + 1
print(freq)  # {'the': 2, 'cat': 1, 'sat': 1, 'on': 1, 'mat': 1}
```
