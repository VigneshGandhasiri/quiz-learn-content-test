## What Are Lists? 📦📦📦

A **list** is an ordered collection of items stored in a single variable.

Think of it like a **train** 🚂 — each **car** holds a value, and they're all connected in order:
```
Index:    [0]        [1]        [2]        [3]        [4]
         ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐
Train:   │ "A"  │──│ "B"  │──│ "C"  │──│ "D"  │──│ "E"  │
         └──────┘  └──────┘  └──────┘  └──────┘  └──────┘
```

## Creating Lists 🛠️

```python
# Empty list
empty = []

# List of numbers
scores = [85, 92, 78, 95, 88]

# List of strings
fruits = ["apple", "banana", "cherry"]

# Mixed types (allowed but uncommon)
mixed = ["Alex", 14, True, 3.14]

# List from range
numbers = list(range(1, 6))  # [1, 2, 3, 4, 5]
```

## Indexing & Accessing Elements 🔍

Lists are **zero-indexed** — the first element is at position 0!

```python
fruits = ["apple", "banana", "cherry", "date"]
#          [0]       [1]       [2]       [3]

print(fruits[0])    # "apple"
print(fruits[2])    # "cherry"
print(fruits[-1])   # "date" (last element!)
print(fruits[-2])   # "cherry" (second from last)
```

### Slicing
```python
numbers = [10, 20, 30, 40, 50, 60]

print(numbers[1:4])    # [20, 30, 40]  (index 1 to 3)
print(numbers[:3])     # [10, 20, 30]  (first 3)
print(numbers[3:])     # [40, 50, 60]  (from index 3 onward)
print(numbers[::2])    # [10, 30, 50]  (every 2nd element)
```

## Modifying Lists ✏️

```python
colors = ["red", "green", "blue"]

# Change an element
colors[1] = "yellow"         # ["red", "yellow", "blue"]

# Add to end
colors.append("purple")      # ["red", "yellow", "blue", "purple"]

# Insert at position
colors.insert(0, "white")    # ["white", "red", "yellow", "blue", "purple"]

# Remove by value
colors.remove("yellow")      # ["white", "red", "blue", "purple"]

# Remove by index
deleted = colors.pop(1)      # Removes and returns "red"

# Length
print(len(colors))           # 3
```

## Looping Through Lists 🔄

### For-each loop (most common)
```python
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I like {fruit}!")
```

### With index
```python
scores = [85, 92, 78, 95, 88]
for i in range(len(scores)):
    print(f"Student {i+1}: {scores[i]}")
```

### Building a list with a loop
```python
squares = []
for i in range(1, 6):
    squares.append(i ** 2)
print(squares)  # [1, 4, 9, 16, 25]
```

## Common List Operations 📊

```python
numbers = [3, 1, 4, 1, 5, 9, 2, 6]

print(len(numbers))      # 8
print(min(numbers))      # 1
print(max(numbers))      # 9
print(sum(numbers))      # 31
print(sorted(numbers))   # [1, 1, 2, 3, 4, 5, 6, 9]
print(5 in numbers)      # True
print(7 in numbers)      # False
```
