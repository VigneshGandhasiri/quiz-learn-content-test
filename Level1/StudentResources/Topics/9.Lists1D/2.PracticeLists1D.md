# Lists (1D): Practice Exercises ✏️

Work with Python lists — creation, access, modification, and iteration!

---

## 1. List Basics

```python
fruits = ["apple", "banana", "cherry"]
fruits.append("date")
print(fruits)         # ['apple', 'banana', 'cherry', 'date']
print(fruits[1])      # banana
print(len(fruits))    # 4
```

---

## 2. List Iteration

```python
scores = [85, 92, 78, 95, 88]
total = 0
for score in scores:
    total += score
avg = total / len(scores)
print(f"Average: {avg:.2f}")
```

## Practice Exercises

1. **Grocery List Builder** — Start with an empty list. Let the user add items. Print the list after each addition. Stop when they type "done".

2. **Find the Max** — Given a list of numbers, find the maximum WITHOUT using the built-in `max()` function.

3. **Remove Duplicates** — Given a list, create a new list with duplicates removed while keeping the original order.

4. **Rotate Left** — Given a list and a number k, rotate the list k positions to the left. For example: [1,2,3,4,5] rotated left by 2 → [3,4,5,1,2].

5. **Merge Two Sorted Lists** — Given two sorted lists, merge them into one sorted list without using `sort()`.

## Challenge Exercises 🌟

6. **List Compression** — Given a sorted list, compress consecutive numbers into ranges. For example: [1,2,3,5,7,8,9] → "1-3, 5, 7-9".

7. **Frequency Count** — Count how many times each element appears in a list using only lists (no dictionaries).
