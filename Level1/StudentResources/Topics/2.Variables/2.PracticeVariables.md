# Variables: Practice Exercises ✏️

Practice creating, updating, and tracing variables!

---

## 1. Basic Variable Creation

```python
pet_name = "Buddy"
pet_age = 5
print(pet_name, "is", pet_age, "years old")
```

---

## 2. Updating a Score

```python
score = 0
score = score + 10  # Earned 10 points
score += 25         # Earned 25 more
score -= 5          # Lost 5 points
print("Final score:", score)  # 30
```

---

## 3. Simple Swap

```python
a = "left"
b = "right"
a, b = b, a
print("a =", a)  # right
print("b =", b)  # left
```

## Practice Exercises

1. **Student Profile** — Create variables for your name, age, school, and favorite subject. Print them in a formatted sentence.

2. **Price Update** — Create a variable `price = 100`. Apply a 20% discount and store the new price. Print both old and new prices.

3. **Tracing Challenge** — What does this code print? Figure it out WITHOUT running it first!
   ```python
   a = 5
   b = 10
   a = a + b
   b = a - b
   a = a - b
   print("a =", a, "b =", b)
   ```

4. **Counter** — Start a variable `count` at 0. Add 1 to it five times using `count += 1`. Print the count after each addition.

5. **Temperature Converter** — Store a temperature in Celsius. Convert to Fahrenheit using `F = C * 9/5 + 32`. Print both values.

---

## Challenge Exercises 🌟

6. **Three-Way Swap** — You have three variables `a = 1`, `b = 2`, `c = 3`. Swap them so `a = 3`, `b = 1`, `c = 2` using a temp variable.

7. **Running Total** — Ask the user for 3 numbers one at a time. Keep a running total and print it after each input.

8. **Variable Detective** — Which of these are valid Python variable names? `myAge`, `2fast`, `_secret`, `my-name`, `class`, `TOTAL_SCORE`
