## Decision Making 🤔

Every day you make decisions based on conditions:
- **IF** it's raining → bring an umbrella
- **IF** you're hungry → eat lunch, **ELSE** → keep working
- **IF** score ≥ 90 → Grade A, **ELIF** score ≥ 80 → Grade B, **ELSE** → keep studying

### 🧭 Maze Runner Analogy 🏃

Imagine running through a maze:
- At every fork, you check a condition: "Is there a wall to the left?"
- **IF wall** → turn right
- **ELSE** → go left
- Each decision changes your path through the maze!

## if Statements ✅

The simplest conditional: do something **only if** a condition is true.

```python
age = 16

if age >= 13:
    print("You can create a social media account!")
```

### ⚠️ Indentation Is Critical!
Python uses **indentation** (4 spaces) to define what's "inside" the if block:

```python
temperature = 35

if temperature > 30:
    print("It's hot outside!")      # ← Inside the if (indented)
    print("Wear sunscreen!")        # ← Also inside the if
print("Have a nice day!")           # ← Outside the if (always runs)
```

## if-else Statements 🔀

Choose between **two paths**: one if the condition is true, another if it's false.

```python
age = 10

if age >= 18:
    print("You can vote!")
else:
    print("You're too young to vote.")
    years_left = 18 - age
    print(f"Come back in {years_left} years!")
```

### 🧭 Bouncer Rule Analogy 🚪

Imagine a bouncer at a club:
- **IF** wearing sneakers → "Come in!"
- **ELSE** → "Sorry, dress shoes only!"

```python
shoes = input("What shoes are you wearing? ")

if shoes == "sneakers":
    print("Welcome in!")
else:
    print("Sorry, sneakers only!")
```

## if-elif-else Chains ⛓️

When you have **multiple conditions** to check in order:

```python
score = int(input("Enter your score: "))

if score >= 90:
    print("Grade: A — Excellent!")
elif score >= 80:
    print("Grade: B — Good job!")
elif score >= 70:
    print("Grade: C — You passed!")
elif score >= 60:
    print("Grade: D — Needs improvement.")
else:
    print("Grade: F — Study harder!")
```

### Flowchart View
```
          score ≥ 90?
         /          \
       YES          NO
        |            |
    Grade A    score ≥ 80?
                /        \
              YES        NO
               |          |
           Grade B   score ≥ 70?
                       /      \
                     YES      NO
                      |        |
                  Grade C    ...
```

### 🎯 Important: Order Matters!
`elif` conditions are checked **top to bottom**. Once one matches, the rest are **skipped**.

## Comparison Operators 📊

| Operator | Meaning | Example | Result |
|----------|---------|---------|--------|
| `==` | Equal to | `5 == 5` | `True` |
| `!=` | Not equal to | `5 != 3` | `True` |
| `>` | Greater than | `10 > 5` | `True` |
| `<` | Less than | `3 < 7` | `True` |
| `>=` | Greater or equal | `5 >= 5` | `True` |
| `<=` | Less or equal | `4 <= 3` | `False` |

```python
x = 10
print(x == 10)    # True
print(x != 5)     # True
print(x > 20)     # False
print(x <= 10)    # True
```

## Boolean Logic: and, or, not 🔗

Combine multiple conditions together!

### `and` — Both must be true
```python
age = 16
has_permit = True

if age >= 16 and has_permit:
    print("You can drive!")
```

### `or` — At least one must be true
```python
day = "Saturday"

if day == "Saturday" or day == "Sunday":
    print("It's the weekend!")
```

### `not` — Flips True to False (and vice versa)
```python
is_raining = False

if not is_raining:
    print("No umbrella needed!")
```

### Truth Tables

| A | B | A and B | A or B |
|---|---|---------|--------|
| True | True | True | True |
| True | False | False | True |
| False | True | False | True |
| False | False | False | False |

## Nested Conditionals 🪆

You can put `if` statements **inside** other `if` statements:

```python
age = int(input("Enter your age: "))

if age >= 0:
    if age < 13:
        print("You are a child.")
    elif age < 20:
        print("You are a teenager.")
    else:
        print("You are an adult.")
else:
    print("Error: Age cannot be negative!")
```

## 🎯 Key Takeaways

| Concept | Purpose | Syntax |
|---------|---------|--------|
| `if` | Do something if condition is true | `if x > 5:` |
| `else` | Do something if condition is false | `else:` |
| `elif` | Check another condition | `elif x > 3:` |
| `and` | Both conditions must be true | `if a and b:` |
| `or` | At least one must be true | `if a or b:` |
| `not` | Flip the condition | `if not done:` |
