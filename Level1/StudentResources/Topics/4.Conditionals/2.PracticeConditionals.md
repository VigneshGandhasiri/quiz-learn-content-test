# Conditionals: Practice Exercises ✏️

Practice making decisions in your code with if, elif, and else!

---

## 1. Even or Odd

```python
number = int(input("Enter a number: "))
if number % 2 == 0:
    print("Even!")
else:
    print("Odd!")
```

---

## 2. Ticket Price

```python
age = int(input("Your age: "))
if age < 5:
    print("Free!")
elif age <= 12:
    print("Child ticket: $5")
elif age <= 64:
    print("Adult ticket: $10")
else:
    print("Senior ticket: $7")
```

## Practice Exercises

1. **Login System** — Ask for a username and password. If username is "admin" AND password is "1234", print "Access granted!" Otherwise print "Access denied!"

2. **Number Classifier** — Given a number, print whether it is positive, negative, or zero.

3. **Leap Year Checker** — A year is a leap year if divisible by 4, BUT not by 100, UNLESS also divisible by 400. Check if a given year is a leap year.

4. **BMI Calculator** — Ask for weight (kg) and height (m). Calculate BMI = weight / height². Classify: Underweight (<18.5), Normal (18.5-24.9), Overweight (25-29.9), Obese (30+).

5. **Rock Paper Scissors** — Ask two players for their choices. Determine the winner using if/elif/else.
