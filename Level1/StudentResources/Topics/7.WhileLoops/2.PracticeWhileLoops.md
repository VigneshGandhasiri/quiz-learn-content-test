# While Loops: Practice Exercises ✏️

Practice condition-driven repetition!

---

## 1. Countdown Timer

```python
seconds = 10
while seconds > 0:
    print(f"{seconds}...")
    seconds -= 1
print("Time's up! ⏰")
```

---

## 2. Password Gate

```python
password = ""
while password != "open sesame":
    password = input("Enter the password: ")
print("Door opened! 🚪")
```

## Practice Exercises

1. **Sum Until Negative** — Keep asking for numbers. When the user enters a negative number, stop and print the sum of all positive numbers entered.

2. **Doubling Pennies** — Start with 1 penny. Keep doubling. Print each day's amount and stop when you have more than $1,000,000.

3. **Collatz Sequence** — Start with any positive number. If even, divide by 2. If odd, multiply by 3 and add 1. Repeat until you reach 1. Print each step.

4. **Number Guesser** — Generate a random number 1-100. Let the user guess with "too high"/"too low" hints. Count guesses.

5. **ATM Machine** — Start with balance $1000. Let users withdraw or deposit. Show balance after each transaction. Quit when they choose to exit.

---

## Challenge Exercises 🌟

6. **Prime Checker** — Ask for a number and use a while loop to check if it's prime.

7. **Digital Root** — Keep summing the digits of a number until you get a single digit (e.g., 493 → 4+9+3=16 → 1+6=7).
