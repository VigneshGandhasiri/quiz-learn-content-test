# 🔁 While Loops — Repeat Until Done

Sometimes you don't know **how many times** to repeat — you just know **when to stop**. That's where `while` loops shine! They keep going as long as a condition is true.

## For vs While 🆚

| Feature | `for` loop | `while` loop |
|---------|-----------|-------------|
| When to use | Know how many times | Don't know how many |
| Control | Counter/sequence | Condition |
| Risk | None | Infinite loop possible! |
| Example | Count 1-10 | Ask until correct answer |

---

## Basic While Loop 📝

```python
count = 1
while count <= 5:
    print(count)
    count += 1   # Don't forget this!
```
**Output:**
```
1
2
3
4
5
```

### 🧭 Filling Water Cups Analogy 🥤

Imagine filling cups with water:
- **WHILE** there are empty cups → fill the next one
- When all cups are full → stop!

```python
cups_filled = 0
total_cups = 5

while cups_filled < total_cups:
    cups_filled += 1
    print(f"Cup {cups_filled} filled! 🥤")

print("All cups filled! ✅")
```

## Loop Termination ⛔

### ⚠️ The #1 Rule: Every while loop MUST eventually become False!

```python
# ✅ GOOD — counter changes, loop will end
i = 10
while i > 0:
    print(i)
    i -= 1    # i gets smaller each time

# ❌ BAD — Infinite loop! Nothing changes!
# x = 5
# while x > 0:
#     print(x)   # x never changes, so x > 0 is ALWAYS true!
```

### 🧭 Melting Tower Analogy 🧊

Imagine a tower of ice blocks in the sun:
- Each round, one block melts away
- **WHILE** blocks remaining > 0 → tower still stands
- Eventually all blocks melt → loop ends!

```python
blocks = 5
while blocks > 0:
    print(f"Tower has {blocks} blocks: " + "🧊" * blocks)
    blocks -= 1
print("Tower has melted! 💧")
```
**Output:**
```
Tower has 5 blocks: 🧊🧊🧊🧊🧊
Tower has 4 blocks: 🧊🧊🧊🧊
Tower has 3 blocks: 🧊🧊🧊
Tower has 2 blocks: 🧊🧊
Tower has 1 blocks: 🧊
Tower has melted! 💧
```

## Common Patterns 📋

### Input Validation Loop
```python
age = int(input("Enter your age (0-120): "))

while age < 0 or age > 120:
    print("Invalid age! Try again.")
    age = int(input("Enter your age (0-120): "))

print(f"Your age is {age}.")
```

### Guessing Game
```python
secret = 42
guess = int(input("Guess the number: "))

while guess != secret:
    if guess < secret:
        print("Too low!")
    else:
        print("Too high!")
    guess = int(input("Guess again: "))

print("Correct! 🎉")
```

### Sentinel Value (Keep going until a special value)
```python
total = 0
number = int(input("Enter a number (0 to stop): "))

while number != 0:
    total += number
    number = int(input("Enter a number (0 to stop): "))

print(f"Total: {total}")
```

## Infinite Loops & break 🛑

### Using `break` to Exit Early
```python
while True:    # Intentional infinite loop
    command = input("Enter command (quit to exit): ")
    if command == "quit":
        break  # Exit the loop immediately!
    print(f"You typed: {command}")

print("Goodbye!")
```

### Using `continue` to Skip
```python
for i in range(1, 11):
    if i % 3 == 0:
        continue   # Skip multiples of 3
    print(i)
# Output: 1 2 4 5 7 8 10
```

## 🎯 Key Takeaways

| Concept | Description | Example |
|---------|-------------|---------|
| `while` | Repeat while condition is true | `while x > 0:` |
| Termination | Loop must eventually end | Update the condition variable! |
| `break` | Exit loop immediately | `if done: break` |
| `continue` | Skip to next iteration | `if skip: continue` |
| Sentinel | Special value to stop input | `while x != 0:` |
