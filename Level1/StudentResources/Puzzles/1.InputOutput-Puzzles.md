## 🟢 Puzzle 1: Pizza Oven 🍕 (⭐ Beginner)

You're running a pizza shop! Your oven is a **black box**: raw ingredients go IN, finished pizza comes OUT.

**Task:** Write a program that:
1. Asks the user for a **pizza size** (small, medium, large)
2. Asks for a **topping**
3. Prints the finished pizza order

**Example:**
```
Enter pizza size: medium
Enter topping: pepperoni
🍕 One MEDIUM pepperoni pizza coming right up!
Order total: $12.99
```

Prices: small = $8.99, medium = $12.99, large = $16.99

<details>
<summary>💡 Answer</summary>

```python
size = input("Enter pizza size: ")
topping = input("Enter topping: ")

print(f"🍕 One {size.upper()} {topping} pizza coming right up!")

if size == "small":
    print("Order total: $8.99")
elif size == "medium":
    print("Order total: $12.99")
else:
    print("Order total: $16.99")
```

</details>

## 🟢 Puzzle 2: Alien Translator 👽 (⭐ Beginner)

An alien species communicates by **reversing words** and adding "ZOP" at the end.

**Task:** Write a program that:
1. Asks the user for an English word
2. Reverses it
3. Adds "ZOP" at the end
4. Prints the "alien translation"

**Example:**
```
Enter a word: hello
Alien translation: ollehZOP
```

<details>
<summary>💡 Answer</summary>

```python
word = input("Enter a word: ")
alien_word = word[::-1] + "ZOP"
print(f"Alien translation: {alien_word}")
```

</details>

## 🟡 Puzzle 3: Recipe Scaler 🧑‍🍳 (⭐⭐ Intermediate)

A recipe serves 4 people. The user wants to cook for a different number.

**Task:** Ask for the desired number of servings. Scale the recipe:
- 2 cups flour
- 3 eggs
- 1.5 cups milk
- 0.5 cups sugar

**Example:**
```
How many servings? 8
For 8 servings you need:
Flour: 4.0 cups
Eggs: 6.0
Milk: 3.0 cups
Sugar: 1.0 cups
```

<details>
<summary>💡 Answer</summary>

```python
servings = int(input("How many servings? "))
scale = servings / 4

print(f"For {servings} servings you need:")
print(f"Flour: {2 * scale} cups")
print(f"Eggs: {3 * scale}")
print(f"Milk: {1.5 * scale} cups")
print(f"Sugar: {0.5 * scale} cups")
```

</details>

## 🔴 Puzzle 4: Secret Encoder 🔐 (⭐⭐⭐ Advanced)

Build a simple encoder that shifts each letter's position by a user-given number (Caesar cipher).

**Task:**
1. Ask for a message and a shift number
2. Shift each letter by that amount
3. Print the encoded message

<details>
<summary>💡 Answer</summary>

```python
message = input("Enter message: ")
shift = int(input("Shift by: "))

encoded = ""
for char in message:
    if char.isalpha():
        base = ord('A') if char.isupper() else ord('a')
        encoded += chr((ord(char) - base + shift) % 26 + base)
    else:
        encoded += char

print(f"Encoded: {encoded}")
```

</details>
