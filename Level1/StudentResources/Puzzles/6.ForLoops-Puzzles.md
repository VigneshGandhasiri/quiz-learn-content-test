## 🟢 Puzzle 1: Counting Stars ⭐ (⭐ Beginner)

**Problem:** Print a right triangle of stars where row `i` has `i` stars.

```
*
**
***
****
*****
```

**Bonus:** Now print it upside down!

<details>
<summary>💡 Answer</summary>

**Right triangle:**
```python
for i in range(1, 6):
    print("*" * i)
```

**Upside down:**
```python
for i in range(5, 0, -1):
    print("*" * i)
```

</details>

## 🟢 Puzzle 2: Loop Detective 🔍 (⭐ Beginner)

Without running the code, what does each loop print?

**Loop A:**
```python
for i in range(5):
    print(i * 2, end=" ")
```

**Loop B:**
```python
for i in range(10, 0, -2):
    print(i, end=" ")
```

**Loop C:**
```python
total = 0
for i in range(1, 11):
    total += i
print(total)
```

<details>
<summary>💡 Answer</summary>

- **Loop A:** `0 2 4 6 8`
- **Loop B:** `10 8 6 4 2`
- **Loop C:** `55` (sum of 1 through 10)

</details>

## 🟡 Puzzle 3: Staircase Builder 🪜 (⭐⭐ Intermediate)

Print a **right-aligned** staircase of height `n`:

For n = 5:
```
    *
   **
  ***
 ****
*****
```

<details>
<summary>💡 Answer</summary>

```python
n = 5
for i in range(1, n + 1):
    spaces = " " * (n - i)
    stars = "*" * i
    print(spaces + stars)
```

</details>

## 🔴 Puzzle 4: Diamond Pattern 💎 (⭐⭐⭐ Advanced)

Print a diamond of height `2n - 1`:

For n = 4:
```
   *
  ***
 *****
*******
 *****
  ***
   *
```

<details>
<summary>💡 Answer</summary>

```python
n = 4
# Top half (including middle)
for i in range(1, n + 1):
    spaces = " " * (n - i)
    stars = "*" * (2 * i - 1)
    print(spaces + stars)

# Bottom half
for i in range(n - 1, 0, -1):
    spaces = " " * (n - i)
    stars = "*" * (2 * i - 1)
    print(spaces + stars)
```

</details>

## 🌟 Puzzle 5: Multiplication Table Generator (⭐⭐⭐⭐ Expert)

Generate a formatted n×n multiplication table:

```
   1   2   3   4   5
   2   4   6   8  10
   3   6   9  12  15
   4   8  12  16  20
   5  10  15  20  25
```

<details>
<summary>💡 Answer</summary>

```python
n = 5
for i in range(1, n + 1):
    for j in range(1, n + 1):
        print(f"{i*j:4}", end="")
    print()
```

</details>
