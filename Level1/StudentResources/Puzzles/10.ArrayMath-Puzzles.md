## 🟢 Puzzle 1: Temperature Tracker 🌡️ (⭐ Beginner)

A weather station recorded these temperatures over a week:

```python
temps = [72, 68, 75, 80, 77, 71, 69]
days = ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]
```

**Task:** Without using built-in `min()`, `max()`, or `sum()`, find:
1. The hottest day and temperature
2. The coldest day and temperature
3. The average temperature
4. How many days were above average

<details>
<summary>💡 Answer</summary>

```python
temps = [72, 68, 75, 80, 77, 71, 69]
days = ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]

# Find max
hottest = temps[0]
hottest_day = 0
for i in range(len(temps)):
    if temps[i] > hottest:
        hottest = temps[i]
        hottest_day = i

# Find min
coldest = temps[0]
coldest_day = 0
for i in range(len(temps)):
    if temps[i] < coldest:
        coldest = temps[i]
        coldest_day = i

# Find average
total = 0
for t in temps:
    total += t
average = total / len(temps)

# Count above average
above = 0
for t in temps:
    if t > average:
        above += 1

print(f"Hottest: {days[hottest_day]} at {hottest}°F")
print(f"Coldest: {days[coldest_day]} at {coldest}°F")
print(f"Average: {average:.1f}°F")
print(f"Days above average: {above}")
```

**Output:**
- Hottest: Thu at 80°F
- Coldest: Tue at 68°F
- Average: 73.1°F
- Days above average: 3

</details>

## 🟡 Puzzle 2: Running Sum 🏃 (⭐⭐ Intermediate)

Given scores: `[10, 20, 30, 40, 50]`

Create a **running sum** array where each element is the sum of all elements up to that index.

Expected: `[10, 30, 60, 100, 150]`

<details>
<summary>💡 Answer</summary>

```python
scores = [10, 20, 30, 40, 50]
running = [0] * len(scores)
running[0] = scores[0]

for i in range(1, len(scores)):
    running[i] = running[i-1] + scores[i]

print(running)  # [10, 30, 60, 100, 150]
```

</details>

## 🔴 Puzzle 3: Grade Curve 📊 (⭐⭐⭐ Advanced)

A teacher wants to "curve" grades: find the highest score and add the difference between 100 and that score to ALL grades. Then count how many pass (≥ 60).

```python
grades = [45, 72, 88, 36, 91, 67, 53, 78]
```

<details>
<summary>💡 Answer</summary>

```python
grades = [45, 72, 88, 36, 91, 67, 53, 78]

# Find max manually
highest = grades[0]
for g in grades:
    if g > highest:
        highest = g

curve = 100 - highest  # 100 - 91 = 9

# Apply curve
curved = [0] * len(grades)
passing = 0
for i in range(len(grades)):
    curved[i] = grades[i] + curve
    if curved[i] >= 60:
        passing += 1

print(f"Curve: +{curve} points")
print(f"Curved grades: {curved}")
print(f"Passing: {passing}/{len(grades)}")
```

**Output:** Curve +9 → [54, 81, 97, 45, 100, 76, 62, 87] → 6/8 passing

</details>
