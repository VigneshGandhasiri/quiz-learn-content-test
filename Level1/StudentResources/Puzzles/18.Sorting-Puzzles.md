## 🟢 Puzzle 1: Card Sorter 🃏 (⭐ Beginner)

You have cards: `[7, 2, 9, 4, 1]`

**Task:** Sort them using **Selection Sort** by hand. Show the array after each step.

At each step, find the smallest remaining card and swap it to the front.

<details>
<summary>💡 Answer</summary>

```
Start:    [7, 2, 9, 4, 1]
Pass 1:   [1, 2, 9, 4, 7]  ← swapped 1 and 7
Pass 2:   [1, 2, 9, 4, 7]  ← 2 already in place
Pass 3:   [1, 2, 4, 9, 7]  ← swapped 4 and 9
Pass 4:   [1, 2, 4, 7, 9]  ← swapped 7 and 9
Done:     [1, 2, 4, 7, 9]
```

```python
cards = [7, 2, 9, 4, 1]

for i in range(len(cards)):
    min_idx = i
    for j in range(i + 1, len(cards)):
        if cards[j] < cards[min_idx]:
            min_idx = j
    cards[i], cards[min_idx] = cards[min_idx], cards[i]
    print(f"Pass {i+1}: {cards}")
```

## 🔴 Puzzle 3: Insertion Sort on Nearly Sorted Data ⚡ (⭐⭐⭐ Advanced)

**Insertion Sort** is fastest on nearly sorted data. Trace it on:

```python
arr = [1, 2, 5, 3, 4, 6]  # Only 3,4,5 are out of place
```

<details>
<summary>💡 Answer</summary>

```
i=1: key=2, compare with 1 → no shift → [1, 2, 5, 3, 4, 6]
i=2: key=5, compare with 2 → no shift → [1, 2, 5, 3, 4, 6]
i=3: key=3, compare with 5 → shift 5 right → [1, 2, _, 5, 4, 6]
     compare 3 with 2 → no shift → insert 3 → [1, 2, 3, 5, 4, 6]
i=4: key=4, compare with 5 → shift 5 right → [1, 2, 3, _, 5, 6]
     compare 4 with 3 → no shift → insert 4 → [1, 2, 3, 4, 5, 6]
i=5: key=6, compare with 5 → no shift → [1, 2, 3, 4, 5, 6]
```

Only 2 shifts needed! Insertion sort is O(n) for nearly sorted data.

```python
arr = [1, 2, 5, 3, 4, 6]
shifts = 0

for i in range(1, len(arr)):
    key = arr[i]
    j = i - 1
    while j >= 0 and arr[j] > key:
        arr[j + 1] = arr[j]
        j -= 1
        shifts += 1
    arr[j + 1] = key
    print(f"i={i}: {arr}")

print(f"Total shifts: {shifts}")
```

</details>

## 🌟 Puzzle 4: Sort Race 🏁 (⭐⭐⭐⭐ Expert)

Count comparisons for all 3 sorting algorithms on the same input and see which "wins":

```python
data = [64, 34, 25, 12, 22, 11, 90]
```

<details>
<summary>💡 Answer</summary>

```python
import copy

def bubble_sort_count(arr):
    a = copy.copy(arr)
    comps = 0
    n = len(a)
    for i in range(n):
        for j in range(0, n - i - 1):
            comps += 1
            if a[j] > a[j + 1]:
                a[j], a[j + 1] = a[j + 1], a[j]
    return comps

def selection_sort_count(arr):
    a = copy.copy(arr)
    comps = 0
    n = len(a)
    for i in range(n):
        min_idx = i
        for j in range(i + 1, n):
            comps += 1
            if a[j] < a[min_idx]:
                min_idx = j
        a[i], a[min_idx] = a[min_idx], a[i]
    return comps

def insertion_sort_count(arr):
    a = copy.copy(arr)
    comps = 0
    for i in range(1, len(a)):
        key = a[i]
        j = i - 1
        while j >= 0:
            comps += 1
            if a[j] > key:
                a[j + 1] = a[j]
                j -= 1
            else:
                break
        a[j + 1] = key
    return comps

data = [64, 34, 25, 12, 22, 11, 90]
print(f"Bubble Sort:    {bubble_sort_count(data)} comparisons")
print(f"Selection Sort: {selection_sort_count(data)} comparisons")
print(f"Insertion Sort: {insertion_sort_count(data)} comparisons")
```

All around 21 comparisons for this small input — differences show with larger data!

</details>
