## 🟢 Puzzle 1: Reverse the Train 🚂 (⭐ Beginner)

A toy train has numbered cars: `[1, 2, 3, 4, 5]`

The turntable reverses the order. Write code to reverse the list **without** using `.reverse()` or slicing `[::-1]`.

<details>
<summary>💡 Answer</summary>

```python
train = [1, 2, 3, 4, 5]

left = 0
right = len(train) - 1

while left < right:
    # Swap
    train[left], train[right] = train[right], train[left]
    left += 1
    right -= 1

print(train)  # [5, 4, 3, 2, 1]
```

</details>

## 🟡 Puzzle 2: Palindrome Challenge 🔤 (⭐⭐ Intermediate)

Use the **two-pointer technique** to check if a word is a palindrome (reads the same forwards and backwards).

Test with: "racecar", "hello", "madam", "python"

<details>
<summary>💡 Answer</summary>

```python
def is_palindrome(word):
    left = 0
    right = len(word) - 1
    
    while left < right:
        if word[left] != word[right]:
            return False
        left += 1
        right -= 1
    return True

words = ["racecar", "hello", "madam", "python"]
for w in words:
    result = "✅ Yes" if is_palindrome(w) else "❌ No"
    print(f"{w}: {result}")
```

**Output:**
- racecar: ✅ Yes
- hello: ❌ No
- madam: ✅ Yes
- python: ❌ No

</details>

## 🟡 Puzzle 3: Two Sum Finder 🎯 (⭐⭐ Intermediate)

Given a **sorted** list and a target sum, use two pointers to find a pair that adds up to the target.

```python
numbers = [1, 3, 5, 7, 9, 11, 15]
target = 12
```

Find two numbers that sum to 12.

<details>
<summary>💡 Answer</summary>

```python
numbers = [1, 3, 5, 7, 9, 11, 15]
target = 12

left = 0
right = len(numbers) - 1

while left < right:
    current_sum = numbers[left] + numbers[right]
    
    if current_sum == target:
        print(f"Found: {numbers[left]} + {numbers[right]} = {target}")
        break
    elif current_sum < target:
        left += 1
    else:
        right -= 1
```

**Output:** Found: 1 + 11 = 12 (also 3 + 9 and 5 + 7)

</details>

## 🔴 Puzzle 4: Dutch National Flag 🇳🇱 (⭐⭐⭐ Advanced)

Sort an array of 0s, 1s, and 2s in one pass using three pointers!

```python
arr = [2, 0, 1, 2, 0, 1, 0, 2, 1]
# Desired: [0, 0, 0, 1, 1, 1, 2, 2, 2]
```

<details>
<summary>💡 Answer</summary>

```python
arr = [2, 0, 1, 2, 0, 1, 0, 2, 1]

low = 0       # boundary for 0s
mid = 0       # current element
high = len(arr) - 1  # boundary for 2s

while mid <= high:
    if arr[mid] == 0:
        arr[low], arr[mid] = arr[mid], arr[low]
        low += 1
        mid += 1
    elif arr[mid] == 1:
        mid += 1
    else:  # arr[mid] == 2
        arr[mid], arr[high] = arr[high], arr[mid]
        high -= 1

print(arr)  # [0, 0, 0, 1, 1, 1, 2, 2, 2]
```

</details>
