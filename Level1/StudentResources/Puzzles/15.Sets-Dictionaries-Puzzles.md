## 🟢 Puzzle 1: Duplicate Detector 🔍 (⭐ Beginner)

A teacher collected student IDs but some students checked in twice! Find all duplicate check-ins.

```python
check_ins = [101, 203, 305, 101, 407, 203, 509, 305, 101]
```

**Task:** Use a set to find which IDs appeared more than once.

<details>
<summary>💡 Answer</summary>

```python
check_ins = [101, 203, 305, 101, 407, 203, 509, 305, 101]

seen = set()
duplicates = set()

for student_id in check_ins:
    if student_id in seen:
        duplicates.add(student_id)
    else:
        seen.add(student_id)

print(f"Duplicate check-ins: {duplicates}")
# Output: Duplicate check-ins: {101, 203, 305}
```

## 🟡 Puzzle 2: Counting Keeper 📊 (⭐⭐ Intermediate)

Count the frequency of each word in this sentence, then find the most common word:

```python
sentence = "the cat sat on the mat and the cat played with the mat"
```

<details>
<summary>💡 Answer</summary>

```python
sentence = "the cat sat on the mat and the cat played with the mat"
words = sentence.split()

freq = {}
for word in words:
    if word in freq:
        freq[word] += 1
    else:
        freq[word] = 1

# Find most common
most_common = ""
highest_count = 0
for word, count in freq.items():
    if count > highest_count:
        highest_count = count
        most_common = word

print(f"Frequencies: {freq}")
print(f"Most common: '{most_common}' ({highest_count} times)")
```

**Output:** Most common: 'the' (4 times)

</details>

## 🔴 Puzzle 3: Anagram Checker ✍️ (⭐⭐⭐ Advanced)

Two words are anagrams if they contain the same letters in different order. Use a dictionary to check if two words are anagrams.

Test pairs: ("listen", "silent"), ("hello", "world"), ("race", "care")

<details>
<summary>💡 Answer</summary>

```python
def are_anagrams(word1, word2):
    if len(word1) != len(word2):
        return False
    
    counts = {}
    for char in word1:
        counts[char] = counts.get(char, 0) + 1
    
    for char in word2:
        if char not in counts or counts[char] == 0:
            return False
        counts[char] -= 1
    
    return True

pairs = [("listen", "silent"), ("hello", "world"), ("race", "care")]
for w1, w2 in pairs:
    result = "✅ Yes" if are_anagrams(w1, w2) else "❌ No"
    print(f"'{w1}' & '{w2}': {result}")
```

</details>
