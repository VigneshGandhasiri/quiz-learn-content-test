## 🟢 Puzzle 1: Shopping List Manager 🛒 (⭐ Beginner)

Build a shopping list program that supports:
- Add item
- Remove item
- Check if item exists
- Print all items

```python
shopping = []
```

**Task:** Perform these operations and track the list after each:
1. Add: "Milk", "Bread", "Eggs", "Butter"
2. Check: "Cheese" in list?
3. Remove: "Bread"
4. Add: "Cheese"
5. Print final list and length

<details>
<summary>💡 Answer</summary>

```python
shopping = []

# Step 1: Add items
shopping.append("Milk")
shopping.append("Bread")
shopping.append("Eggs")
shopping.append("Butter")
print(f"After adding: {shopping}")

# Step 2: Check for Cheese
has_cheese = "Cheese" in shopping
print(f"Has Cheese? {has_cheese}")  # False

# Step 3: Remove Bread
shopping.remove("Bread")
print(f"After removing Bread: {shopping}")

# Step 4: Add Cheese
shopping.append("Cheese")
print(f"After adding Cheese: {shopping}")

# Step 5: Final state
print(f"Final list: {shopping}")
print(f"Length: {len(shopping)}")
```

**Final:** ["Milk", "Eggs", "Butter", "Cheese"], Length: 4

</details>

## 🟡 Puzzle 2: Index Gymnastics 🤸 (⭐⭐ Intermediate)

Given: `arr = [10, 20, 30, 40, 50, 60, 70, 80]`

What does each expression return? Solve WITHOUT running:

1. `arr[2]`
2. `arr[-1]`
3. `arr[1:4]`
4. `arr[::2]`
5. `arr[-3:]`
6. `arr[::-1]`
7. `arr[1:6:2]`

<details>
<summary>💡 Answer</summary>

1. `arr[2]` → 30
2. `arr[-1]` → 80
3. `arr[1:4]` → [20, 30, 40]
4. `arr[::2]` → [10, 30, 50, 70]
5. `arr[-3:]` → [60, 70, 80]
6. `arr[::-1]` → [80, 70, 60, 50, 40, 30, 20, 10]
7. `arr[1:6:2]` → [20, 40, 60]

</details>

## 🔴 Puzzle 3: List Rotation 🔄 (⭐⭐⭐ Advanced)

Rotate a list `k` positions to the right WITHOUT using slicing or built-in rotate.

```python
arr = [1, 2, 3, 4, 5]
k = 2
# Expected: [4, 5, 1, 2, 3]
```

<details>
<summary>💡 Answer</summary>

```python
arr = [1, 2, 3, 4, 5]
k = 2

# Method: reverse approach
# 1. Reverse entire array
# 2. Reverse first k elements
# 3. Reverse remaining elements

def reverse_section(a, start, end):
    while start < end:
        a[start], a[end] = a[end], a[start]
        start += 1
        end -= 1

k = k % len(arr)  # handle k > len
reverse_section(arr, 0, len(arr) - 1)  # [5,4,3,2,1]
reverse_section(arr, 0, k - 1)          # [4,5,3,2,1]
reverse_section(arr, k, len(arr) - 1)   # [4,5,1,2,3]

print(arr)  # [4, 5, 1, 2, 3]
```

</details>
