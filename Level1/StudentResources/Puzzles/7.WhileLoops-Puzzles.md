## 🟢 Puzzle 1: Filling Water Cups 🥤 (⭐ Beginner)

A water dispenser fills cups one at a time. It has 1000ml of water. Each cup holds 250ml. 

**Task:** Use a while loop to fill cups until the dispenser is empty. Count how many cups are filled.

<details>
<summary>💡 Answer</summary>

```python
water = 1000
cup_size = 250
cups_filled = 0

while water >= cup_size:
    water -= cup_size
    cups_filled += 1
    print(f"Cup {cups_filled} filled! Water left: {water}ml")

print(f"Total cups filled: {cups_filled}")
```

**Output:** 4 cups filled, 0ml remaining

</details>

## 🟡 Puzzle 2: Melting Tower ❄️🗼 (⭐⭐ Intermediate)

An ice tower starts at height 100. Each hour it melts by **10% of its current height** (not original height!). How many hours until the tower is below height 10?

**Trace the first 5 hours, then write the full loop.**

<details>
<summary>💡 Answer</summary>

**Trace:**
| Hour | Height Before | Melt Amount | Height After |
|------|--------------|-------------|--------------|
| 1 | 100.0 | 10.0 | 90.0 |
| 2 | 90.0 | 9.0 | 81.0 |
| 3 | 81.0 | 8.1 | 72.9 |
| 4 | 72.9 | 7.29 | 65.61 |
| 5 | 65.61 | 6.561 | 59.049 |

```python
height = 100
hours = 0

while height >= 10:
    height = height * 0.9  # loses 10%
    hours += 1

print(f"Tower below 10 after {hours} hours")
print(f"Final height: {height:.2f}")
```

**Answer:** 22 hours (height ≈ 9.85)

</details>

## 🔴 Puzzle 3: Number Guessing Game 🎲 (⭐⭐⭐ Advanced)

Build a guessing game where:
1. A random number 1-100 is chosen
2. Player guesses until correct
3. Program says "Higher!" or "Lower!" after each guess
4. Track number of attempts
5. At the end, rate the player: ≤5 = Expert, ≤10 = Good, >10 = Keep Practicing

<details>
<summary>💡 Answer</summary>

```python
import random

secret = random.randint(1, 100)
attempts = 0

print("I'm thinking of a number 1-100!")

while True:
    guess = int(input("Your guess: "))
    attempts += 1
    
    if guess < secret:
        print("Higher! ⬆️")
    elif guess > secret:
        print("Lower! ⬇️")
    else:
        print(f"🎉 Correct! You got it in {attempts} attempts!")
        break

if attempts <= 5:
    print("Rating: 🏆 Expert!")
elif attempts <= 10:
    print("Rating: 👍 Good!")
else:
    print("Rating: 📚 Keep practicing!")
```

</details>

## 🌟 Puzzle 4: Collatz Conjecture (⭐⭐⭐⭐ Expert)

The Collatz problem: Start with any positive integer. If even, divide by 2. If odd, multiply by 3 and add 1. The conjecture says you'll always reach 1.

Write a program that takes a number and prints the sequence until it reaches 1. Also count the steps.

Example: 6 → 3 → 10 → 5 → 16 → 8 → 4 → 2 → 1 (8 steps)

<details>
<summary>💡 Answer</summary>

```python
n = int(input("Start number: "))
steps = 0
print(n, end="")

while n != 1:
    if n % 2 == 0:
        n = n // 2
    else:
        n = n * 3 + 1
    steps += 1
    print(f" → {n}", end="")

print(f"\nReached 1 in {steps} steps!")
```

</details>
