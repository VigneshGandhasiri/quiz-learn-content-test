## 🟢 Puzzle 1: Vending Machine 🥤 (⭐ Beginner)

Build a vending machine using match-case! The machine has these items:

| Code | Item | Price |
|------|------|-------|
| A1 | Cola | $1.50 |
| A2 | Sprite | $1.50 |
| B1 | Chips | $2.00 |
| B2 | Cookies | $1.75 |
| C1 | Water | $1.00 |

Accept a code, deduct from $5.00 balance, and show change.

<details>
<summary>💡 Answer</summary>

```python
balance = 5.00
code = input("Enter item code: ").upper()

match code:
    case "A1":
        item, price = "Cola", 1.50
    case "A2":
        item, price = "Sprite", 1.50
    case "B1":
        item, price = "Chips", 2.00
    case "B2":
        item, price = "Cookies", 1.75
    case "C1":
        item, price = "Water", 1.00
    case _:
        print("Invalid code!")
        item, price = None, 0

if item:
    change = balance - price
    print(f"Dispensing {item}...")
    print(f"Change: ${change:.2f}")
```

</details>

## 🟡 Puzzle 2: TV Remote 📺 (⭐⭐ Intermediate)

Build a TV remote simulator. The TV starts on channel 1, volume 5.

Supported buttons: `power`, `ch_up`, `ch_down`, `vol_up`, `vol_down`, `mute`, `info`

Keep running until the user presses `power` to turn off.

<details>
<summary>💡 Answer</summary>

```python
channel = 1
volume = 5
is_on = True
muted = False

while is_on:
    button = input("Press button: ").lower()
    
    match button:
        case "power":
            is_on = False
            print("TV turning off... 📺")
        case "ch_up":
            channel = min(channel + 1, 999)
            print(f"Channel: {channel}")
        case "ch_down":
            channel = max(channel - 1, 1)
            print(f"Channel: {channel}")
        case "vol_up":
            volume = min(volume + 1, 10)
            muted = False
            print(f"Volume: {'🔊' * volume}")
        case "vol_down":
            volume = max(volume - 1, 0)
            print(f"Volume: {'🔊' * volume}")
        case "mute":
            muted = not muted
            print("🔇 Muted" if muted else f"🔊 Volume: {volume}")
        case "info":
            print(f"CH: {channel} | VOL: {volume} | {'Muted' if muted else 'Unmuted'}")
        case _:
            print("Unknown button!")
```

</details>
