## 🟢 Puzzle 1: Maze Runner 🏃 (⭐ Beginner)

You're navigating a maze. At each junction, check the condition and determine which way you go:

```python
position = "start"
has_key = True
health = 75
gold = 30

if has_key:
    position = "door_opened"
    
if health > 50:
    position = "fight_monster"
    health = health - 25
else:
    position = "run_away"

if gold >= 50:
    position = "buy_armor"
elif gold >= 20:
    position = "buy_potion"
else:
    position = "continue"

print(position, health)
```

**Question:** What is the final position and health?

<details>
<summary>💡 Answer</summary>

1. `has_key` is True → position = "door_opened"
2. `health` (75) > 50 → position = "fight_monster", health = 50
3. `gold` (30) >= 50? No. `gold` >= 20? Yes → position = "buy_potion"

**Output:** `buy_potion 50`

Note: Each `if` is independent (not elif), so position gets overwritten each time!

</details>

## 🟡 Puzzle 2: Bouncer Rules 🚪 (⭐⭐ Intermediate)

A club has these rules:
- Must be 18+ OR accompanied by a parent
- Must NOT be on the banned list
- Must have proper shoes (sneakers or boots)

```python
age = 16
has_parent = True
is_banned = False
shoes = "sneakers"
```

Write the conditional logic that prints "Welcome!" or "Sorry, can't enter." with a reason.

<details>
<summary>💡 Answer</summary>

```python
age = 16
has_parent = True
is_banned = False
shoes = "sneakers"

if is_banned:
    print("Sorry, you're on the banned list!")
elif age < 18 and not has_parent:
    print("Sorry, must be 18+ or have a parent!")
elif shoes not in ["sneakers", "boots"]:
    print("Sorry, improper footwear!")
else:
    print("Welcome!")
```

**Output:** "Welcome!" (age < 18 but has_parent, not banned, proper shoes)

</details>

## 🔴 Puzzle 3: Nested Decision Tree 🌳 (⭐⭐⭐ Advanced)

A loan approval system:
- Credit score ≥ 700 → Check income
  - Income ≥ $50,000 → APPROVED
  - Income ≥ $30,000 → APPROVED with higher rate
  - Otherwise → DENIED
- Credit score ≥ 500 → Check income AND assets
  - Income ≥ $60,000 AND assets ≥ $10,000 → APPROVED with higher rate
  - Otherwise → DENIED
- Credit score < 500 → DENIED

Implement this and test with: score=650, income=65000, assets=15000

<details>
<summary>💡 Answer</summary>

```python
score = 650
income = 65000
assets = 15000

if score >= 700:
    if income >= 50000:
        print("APPROVED - Standard rate")
    elif income >= 30000:
        print("APPROVED - Higher rate")
    else:
        print("DENIED - Insufficient income")
elif score >= 500:
    if income >= 60000 and assets >= 10000:
        print("APPROVED - Higher rate")
    else:
        print("DENIED - Need higher income and assets")
else:
    print("DENIED - Credit score too low")
```

**Output:** "APPROVED - Higher rate" (score 650 ≥ 500, income 65000 ≥ 60000, assets 15000 ≥ 10000)

</details>
