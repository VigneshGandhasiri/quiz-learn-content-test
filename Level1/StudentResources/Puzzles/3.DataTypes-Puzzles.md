## 🟢 Puzzle 1: Shape Sorter 🔷 (⭐ Beginner)

Match each value to its correct data type container:

| Value | int? | float? | str? | bool? |
|-------|------|--------|------|-------|
| `"42"` | | | | |
| `42` | | | | |
| `42.0` | | | | |
| `True` | | | | |
| `"True"` | | | | |
| `0` | | | | |
| `""` | | | | |
| `3.14` | | | | |

<details>
<summary>💡 Answer</summary>

| Value | Type |
|-------|------|
| `"42"` | str |
| `42` | int |
| `42.0` | float |
| `True` | bool |
| `"True"` | str |
| `0` | int |
| `""` | str (empty string) |
| `3.14` | float |

</details>

## 🟡 Puzzle 2: Menu Data Challenge 🍔 (⭐⭐ Intermediate)

A restaurant menu stores data about items. Assign the correct types:

```python
item_name = ____("Cheeseburger")     # Should be str
item_price = ____("8.99")            # Should be float  
quantity = ____("25")                 # Should be int
is_spicy = ____                      # Should be bool (it IS spicy!)
calories = ____("450")               # Should be int
rating = ____("4.5")                 # Should be float
```

Fill in each blank!

<details>
<summary>💡 Answer</summary>

```python
item_name = str("Cheeseburger")      # str
item_price = float("8.99")           # 8.99
quantity = int("25")                  # 25
is_spicy = True                      # bool
calories = int("450")                # 450
rating = float("4.5")               # 4.5
```

</details>

## 🟡 Puzzle 3: Bug Hunt 🐛 (⭐⭐ Intermediate)

Each of these programs has a **type error**. Find and fix each one!

**Bug 1:**
```python
age = input("Age: ")
can_drive = age >= 16
```

**Bug 2:**
```python
price = 9.99
message = "The price is $" + price
```

**Bug 3:**
```python
total = "100"
tax = total * 0.08
```

<details>
<summary>💡 Answer</summary>

**Bug 1:** `input()` returns a string, can't compare str with int.
```python
age = int(input("Age: "))
can_drive = age >= 16
```

**Bug 2:** Can't concatenate str with float.
```python
message = "The price is $" + str(price)
```

**Bug 3:** `total` is a string, can't multiply str by float.
```python
total = int("100")  # or float("100")
tax = total * 0.08
```

</details>

## 🔴 Puzzle 4: Type Conversion Chain ⛓️ (⭐⭐⭐ Advanced)

What does each step produce? Trace through WITHOUT running:

```python
a = "3.14"
b = float(a)          # b = ?
c = int(b)            # c = ?
d = str(c)            # d = ?
e = bool(c)           # e = ?
f = d * c             # f = ?
g = b + c             # g = ?
print(f, g, e)
```

<details>
<summary>💡 Answer</summary>

1. `b = float("3.14")` → 3.14
2. `c = int(3.14)` → 3 (truncates, doesn't round!)
3. `d = str(3)` → "3"
4. `e = bool(3)` → True (any non-zero number is True)
5. `f = "3" * 3` → "333" (string repetition!)
6. `g = 3.14 + 3` → 6.14
7. **Output:** `333 6.14 True`

</details>
