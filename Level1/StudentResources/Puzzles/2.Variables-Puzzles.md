## 🟢 Puzzle 1: Parking Lot 🅿️ (⭐ Beginner)

A parking lot has 3 spaces (variables). Cars arrive and leave throughout the day.

**Trace through and determine the final state:**
```python
space_A = "Toyota"
space_B = "Honda"
space_C = "empty"

# Toyota leaves, space_A becomes empty
space_C = space_A     # Wait, what happens here?
space_A = "empty"

# New car arrives
space_A = "Tesla"

# Honda and Tesla swap spaces
temp = space_A
space_A = space_B
space_B = temp
```

**Question:** What is in each space at the end?

<details>
<summary>💡 Answer</summary>

- `space_A` = "Honda"
- `space_B` = "Tesla"
- `space_C` = "Toyota"

**Trace:**
1. `space_C = space_A` → space_C = "Toyota"
2. `space_A = "empty"` → space_A = "empty"
3. `space_A = "Tesla"` → space_A = "Tesla"
4. `temp = space_A` → temp = "Tesla"
5. `space_A = space_B` → space_A = "Honda"
6. `space_B = temp` → space_B = "Tesla"

</details>

## 🟢 Puzzle 2: RPG Backpack 🎒 (⭐ Beginner)

Your RPG character has a backpack with 3 slots. You can only carry 3 items — to pick up a new one, you must drop one!

```python
slot_1 = "Sword"
slot_2 = "Shield"
slot_3 = "Potion"

# Find a Magic Staff — drop the Potion to pick it up
slot_3 = "Magic Staff"

# Find Healing Gem — swap with Shield
slot_2, slot_3 = slot_3, slot_2

# Find Dragon Scale — replace oldest item (slot_1)
slot_1 = "Dragon Scale"
```

**Question:** What's in each slot? Which original items were lost?

<details>
<summary>💡 Answer</summary>

- `slot_1` = "Dragon Scale"
- `slot_2` = "Magic Staff"
- `slot_3` = "Shield"

Lost items: Sword and Potion

</details>

## 🟡 Puzzle 3: Number Magician 🎩 (⭐⭐ Intermediate)

**Trace through this code WITHOUT running it:**
```python
x = 10
y = 3
z = x % y        # What is z?
x = x // y       # What is x now?
y = z + x        # What is y now?
x = x ** y       # What is x now?
print(x, y, z)
```

<details>
<summary>💡 Answer</summary>

1. `z = 10 % 3` → z = 1
2. `x = 10 // 3` → x = 3
3. `y = 1 + 3` → y = 4
4. `x = 3 ** 4` → x = 81
5. **Output:** `81 4 1`

</details>

## 🔴 Puzzle 4: The Great Swap Challenge 🔀 (⭐⭐⭐ Advanced)

Swap `a`, `b`, and `c` so that `a` gets `c`'s value, `b` gets `a`'s value, and `c` gets `b`'s value — using only ONE temporary variable.

```python
a = "Apple"
b = "Banana"
c = "Cherry"
# Result should be: a="Cherry", b="Apple", c="Banana"
```

<details>
<summary>💡 Answer</summary>

```python
temp = a        # temp = "Apple"
a = c           # a = "Cherry"
c = b           # c = "Banana"
b = temp        # b = "Apple"
```

Or using Python tuple unpacking:
```python
a, b, c = c, a, b
```

</details>
