## 🟢 Puzzle 1: Lost Teddy 🧸 (⭐ Beginner)

Your teddy bear is hidden in one of these numbered boxes. Use **linear search** to find it!

```python
boxes = ["empty", "socks", "empty", "teddy", "books", "empty", "toys"]
```

**Task:** Search through the boxes one by one. Print which box number (0-indexed) contains the teddy. Count how many boxes you checked.

<details>
<summary>💡 Answer</summary>

```python
boxes = ["empty", "socks", "empty", "teddy", "books", "empty", "toys"]
checks = 0

for i in range(len(boxes)):
    checks += 1
    if boxes[i] == "teddy":
        print(f"🧸 Found teddy in box {i}!")
        print(f"Checked {checks} boxes")
        break
```

**Output:** Found teddy in box 3! Checked 4 boxes.

</details>

## 🟡 Puzzle 2: Super Fast Finder ⚡ (⭐⭐ Intermediate)

Use **binary search** to find a number in a sorted list. Trace through step by step!

```python
numbers = [2, 5, 8, 12, 16, 23, 38, 45, 56, 72, 91]
target = 23
```

**Task:** Trace the binary search:
- What are `left`, `right`, `mid` at each step?
- How many comparisons until found?

<details>
<summary>💡 Answer</summary>

**Trace:**

| Step | left | right | mid | numbers[mid] | Action |
|------|------|-------|-----|-------------|--------|
| 1 | 0 | 10 | 5 | 23 | Found! ✅ |

Found in **1 comparison**! (Lucky — it was at the middle!)

Now try `target = 56`:

| Step | left | right | mid | numbers[mid] | Action |
|------|------|-------|-----|-------------|--------|
| 1 | 0 | 10 | 5 | 23 | 56 > 23, go right |
| 2 | 6 | 10 | 8 | 56 | Found! ✅ |

Found in **2 comparisons**!

```python
numbers = [2, 5, 8, 12, 16, 23, 38, 45, 56, 72, 91]
target = 56

left = 0
right = len(numbers) - 1
comparisons = 0

while left <= right:
    mid = (left + right) // 2
    comparisons += 1
    
    if numbers[mid] == target:
        print(f"Found {target} at index {mid}")
        print(f"Comparisons: {comparisons}")
        break
    elif numbers[mid] < target:
        left = mid + 1
    else:
        right = mid - 1
```

</details>

## 🔴 Puzzle 3: Search Race 🏁 (⭐⭐⭐ Advanced)

Compare linear search vs binary search on a list of 1000 items.

**Task:**
1. Create a sorted list of 1000 numbers
2. Search for the number 997
3. Count comparisons for both methods
4. Which is faster and by how much?

<details>
<summary>💡 Answer</summary>

```python
# Create sorted list
numbers = list(range(1, 1001))
target = 997

# Linear search
linear_comparisons = 0
for i in range(len(numbers)):
    linear_comparisons += 1
    if numbers[i] == target:
        break

# Binary search
binary_comparisons = 0
left, right = 0, len(numbers) - 1
while left <= right:
    mid = (left + right) // 2
    binary_comparisons += 1
    if numbers[mid] == target:
        break
    elif numbers[mid] < target:
        left = mid + 1
    else:
        right = mid - 1

print(f"Linear search: {linear_comparisons} comparisons")
print(f"Binary search: {binary_comparisons} comparisons")
print(f"Binary search was {linear_comparisons // binary_comparisons}x faster!")
```

**Output:**
- Linear: 997 comparisons
- Binary: ~10 comparisons
- Binary was ~100x faster!

</details>
