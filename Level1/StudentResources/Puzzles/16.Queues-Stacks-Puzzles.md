## 🟢 Puzzle 1: Dragon's Queue 🐉 (⭐ Beginner)

Dragons line up to breathe fire at targets. First dragon in line goes first (FIFO)!

```python
dragons = ["Blaze", "Ember", "Frost", "Spark", "Flame"]
```

**Task:** Simulate: each dragon steps up, breathes fire, and leaves the line. Two new dragons ("Storm" and "Shadow") join at the back after 2 rounds.

<details>
<summary>💡 Answer</summary>

```python
dragons = ["Blaze", "Ember", "Frost", "Spark", "Flame"]

for round_num in range(1, 8):
    if not dragons:
        print("No more dragons!")
        break
    
    current = dragons.pop(0)  # dequeue from front
    print(f"Round {round_num}: {current} breathes fire! 🔥")
    
    if round_num == 2:
        dragons.append("Storm")
        dragons.append("Shadow")
        print("  → Storm and Shadow joined the queue!")
    
    print(f"  Queue: {dragons}")
```

**Output:**
- Round 1: Blaze breathes fire!
- Round 2: Ember breathes fire! → Storm and Shadow join
- Round 3: Frost breathes fire!
- Round 4: Spark breathes fire!
- Round 5: Flame breathes fire!
- Round 6: Storm breathes fire!
- Round 7: Shadow breathes fire!

</details>

## 🟡 Puzzle 2: Lunch Line Simulator 🍽️ (⭐⭐ Intermediate)

Students arrive in groups. VIP students (teachers' kids) go to the front. Regular students go to the back.

Simulate this process:
1. ["Alice", "Bob"] arrive (regular)
2. ["Cara"] arrives (VIP - goes to front!)
3. ["Dan", "Eve"] arrive (regular)
4. First 2 students get served
5. ["Fin"] arrives (VIP)

What does the line look like after all events?

<details>
<summary>💡 Answer</summary>

```python
line = []

# Event 1: Alice, Bob arrive (regular)
line.append("Alice")
line.append("Bob")
print(f"After event 1: {line}")

# Event 2: Cara arrives (VIP - front)
line.insert(0, "Cara")
print(f"After event 2: {line}")

# Event 3: Dan, Eve arrive (regular)
line.append("Dan")
line.append("Eve")
print(f"After event 3: {line}")

# Event 4: Serve first 2
served1 = line.pop(0)
served2 = line.pop(0)
print(f"Served: {served1}, {served2}")
print(f"After event 4: {line}")

# Event 5: Fin arrives (VIP)
line.insert(0, "Fin")
print(f"After event 5: {line}")
```

**Final line:** ["Fin", "Dan", "Eve"]

</details>

## 🟡 Puzzle 3: Pancake Tower 🥞 (⭐⭐ Intermediate)

A stack of pancakes — you can only add/remove from the top (LIFO)!

```python
stack = []
```

**Operations:**
1. Cook 3 pancakes (sizes: Small, Medium, Large) — add in order
2. Someone eats the top pancake
3. Cook 2 more (Extra-Large, Jumbo)
4. Someone eats 2 pancakes

What's left on the stack? In what order from top to bottom?

<details>
<summary>💡 Answer</summary>

```python
stack = []

# Step 1: Cook 3 pancakes
stack.append("Small")
stack.append("Medium")
stack.append("Large")
print(f"After cooking 3: {stack}")
# Stack: [Small, Medium, Large] ← Large is on top

# Step 2: Eat top pancake
eaten = stack.pop()
print(f"Ate: {eaten}")
# Stack: [Small, Medium]

# Step 3: Cook 2 more
stack.append("Extra-Large")
stack.append("Jumbo")
print(f"After cooking 2 more: {stack}")
# Stack: [Small, Medium, Extra-Large, Jumbo]

# Step 4: Eat 2 pancakes
eaten1 = stack.pop()  # Jumbo
eaten2 = stack.pop()  # Extra-Large
print(f"Ate: {eaten1}, {eaten2}")

print(f"Remaining (bottom to top): {stack}")
# Stack: [Small, Medium]
```

**Answer:** [Small, Medium] — Medium on top, Small on bottom

</details>

## 🔴 Puzzle 4: Mirror Word Checker 🪞 (⭐⭐⭐ Advanced)

Use a **stack** to check if a word reads the same forwards and backwards (palindrome check using stack approach).

How it works:
1. Push each character onto a stack
2. Pop each character and build the reversed string
3. Compare original to reversed

Test: "racecar", "kayak", "hello"

<details>
<summary>💡 Answer</summary>

```python
def is_mirror_word(word):
    stack = []
    
    # Push all characters
    for char in word:
        stack.append(char)
    
    # Pop all characters to build reverse
    reversed_word = ""
    while stack:
        reversed_word += stack.pop()
    
    return word == reversed_word

words = ["racecar", "kayak", "hello"]
for w in words:
    result = "🪞 Mirror word!" if is_mirror_word(w) else "❌ Not a mirror"
    print(f"{w}: {result}")
```

**Output:**
- racecar: 🪞 Mirror word!
- kayak: 🪞 Mirror word!
- hello: ❌ Not a mirror

</details>

## 🌟 Puzzle 5: Balanced Brackets (⭐⭐⭐⭐ Expert)

Check if a string of brackets is properly balanced using a stack.

Test: `"((()))"`, `"(()("`, `"({[]})"`, `"({[}])"`

<details>
<summary>💡 Answer</summary>

```python
def is_balanced(text):
    stack = []
    pairs = {')': '(', ']': '[', '}': '{'}
    
    for char in text:
        if char in "({[":
            stack.append(char)
        elif char in ")}]":
            if not stack or stack[-1] != pairs[char]:
                return False
            stack.pop()
    
    return len(stack) == 0

tests = ["((()))", "(()(", "({[]})", "({[})"]
for t in tests:
    result = "✅ Balanced" if is_balanced(t) else "❌ Not balanced"
    print(f"{t}: {result}")
```

</details>
