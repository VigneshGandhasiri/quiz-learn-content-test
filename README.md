## 1. Arithmetic Operators

| Operator | Description | Example |
|--------|------------|---------|
| `+` | Addition | `5 + 3 = 8` |
| `-` | Subtraction | `5 - 3 = 2` |
| `*` | Multiplication | `5 * 3 = 15` |
| `/` | Division | `6 / 3 = 2` |
| `%` | Modulus | `5 % 2 = 1` |

```python
a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a / b)
